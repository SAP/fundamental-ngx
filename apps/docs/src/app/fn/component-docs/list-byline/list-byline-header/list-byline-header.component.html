Byline List
This is the byline list, where the list items are taller than the default list and information can be conveyed on two lines. All of the examples from List section apply to byline list too. Use [byline] input on the fn-list and you will get byline mode of fn-list. Then you'll have access to fnListItemByline directive, which will help you to set second line of text, under title section of fn-list-item.
