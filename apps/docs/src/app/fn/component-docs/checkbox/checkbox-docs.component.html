Standard Checkbox By default fn-checkbox component is binary, returning true and false values. Checkbox custom values Values returned by checkbox can be customized using [values] property.
  • Type: {{ '{ trueValue?: any, falseValue?: any }' }}
  • Default value: {{ '{ trueValue: true, falseValue: false }' }}
Checkbox with Reactive Forms Checkbox can be used with Reactive Forms by binding it as standard form control with [formControlName] input. Example below shows usage of checkbox component with Reactive Form. "Accept all" behaves as main control representing overall state of agreements controls.

You can set [isGroup]="true" to make the checkbox take the whole width of the parent container. This way all checkboxes in a group will have the same width. By default, the width of the checkbox is set to "fit-content". Checkbox with custom label Checkbox label can by passed to component as external content using content projection mechanism. If external content is provided value passed using [label] property is ignored.
