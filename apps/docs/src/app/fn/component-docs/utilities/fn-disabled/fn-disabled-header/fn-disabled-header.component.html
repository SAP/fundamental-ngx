Disabled directive

When you apply the [fnDisabled] directive to the element it will be effectively disabled. It will have class .is-disabled, as well as attributes disabled and aria-disabled.

This directive is also provided via injection token fnDisabled through the DI tree, but also, most importantly you can add FnDisabledProvider to your providers list and gain always available watcher on disabled state, whether or not you apply [fnDisabled] to your component.
