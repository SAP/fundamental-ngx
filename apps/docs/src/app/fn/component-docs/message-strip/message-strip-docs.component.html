Message Strip Standard Usage

The message strip displays information bars with semantic colors and icons, indicating different levels of urgency and/or action required by the user.

The message strip is fully responsive, with an icon and close button on opposite sides of the message text.

In order to show icon on the left side of the Message Strip component, developers must pass appropriate icon to [glyph] input property.

Developers have an ability to hide dismiss button by providing [dismissible]="false" input property.
