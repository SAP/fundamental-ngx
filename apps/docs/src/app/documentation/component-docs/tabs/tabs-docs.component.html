Programmatic Selection

There are two ways of accomplishing this. First, the selectedIndex input can be used. This requires maintaining a flag. The second method requires setting a template reference variable and then calling the selectTab(index) method.

Complex Tab Titles

Use an ng-template with the fd-tab-title directive to access heavily customizable tab titles.

Tab Navigation Mode

It is possible to use the tabs in navigation mode. While there is no option to provide some content in this mode, it does allow the use of a router-outlet with [routerLink] or [href] attributes.
