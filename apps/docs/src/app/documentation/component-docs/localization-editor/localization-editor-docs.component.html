Basic Localization Editor

Basic usage of the localization editor. The add-on content on the right-side can be customized through the [label] input.

Forms in Localization Editor

The localization editor component is compatible with Angular forms.

Complex Add-On content

As a more flexible alternative to the [label] input, attaching the fd-localization-editor-label directive to an ng-template allows for complex content to be displayed in the add-on space.

Textarea

The localization component can be used with textarea. Textarea element should be used with fd-localization-editor-textarea directive.
