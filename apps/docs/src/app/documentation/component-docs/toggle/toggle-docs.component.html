Toggle Sizes

Options include 'xs', 's', 'l' and default. If no value is provided, default is used.

Disabled State

Toggle can be in the disabled state, which prevents the user from interacting with it.

Two-Way Binding

It is possible to two-way bind to the checked property of the fd-toggle.
