Button Options There are three emphasis styles used to indicate the importance of the button on the page. Do not provide a value for the default option. Use [options]="'emphasized'" or [options]="'light'" to set the option. Button Types The default button type is "action", no modifier is required. Use [fdType]="'standard'", [fdType]="'positive'", [fdType]="'medium'", or [fdType]="'negative'" to modify the type of the button. Buttons with Icons Use [glyph]="'{{ '{' }}icon-name}'" to change an icon of dropdown menu. Open/Close Programmatically It is possible to control the open state of the popover through two-way binding to the [(isOpen)] property. Custom Template for Action Button It is possible to add a custom template to the action button through the fd-split-button-action-title directive.
