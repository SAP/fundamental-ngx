Basic usage This example shows basic usage of embeding micro app into main app.
  • click inside hero micoapp, it emits event to main app, main app change simple microapp's input parameters in response of the event, simple microapp react input data changes by refreshing the list.
  • click inside both simple and hero microap, main app show events in event log.
  • simple microapp and hero microapp both use same classname 'microapp-container' for outer div with different styles
    simple microapp definition as:
    .microapp-container {{ '{'}}
        margin: 4px;
        padding: 4px;
        border-width: 1px;
        border-style: solid;
    {{ '}' }}
    hero microapp definition as:
    .microapp-container {{ '{'}}
        margin: 10px;
        padding: 10px;
        border-width: 1px;
        border-style: inset;
    {{ '}' }}
    .microapp-container used in both micrapp on same page do not clash each other, they works as design expect.
  • Hero microapp use router inside, it inherit parent(main app) routing root and integrate with main app routing path.
