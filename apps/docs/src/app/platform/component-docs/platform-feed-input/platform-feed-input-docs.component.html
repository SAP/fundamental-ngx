Standard

The standard feed input component displays an empty text field with an inactive submit button. The button would be active when textarea has a value and calls (onSubmit) event when button was pressed

Placeholder

If avatarSrc property not set the user thumb displays with generic image

No avatar

Set avatarShow="false" if you don't need use user image

Disabled

Set disabled="true" to disable text field with submit button.

Max Height

The max height sets row count in maxHeight property. Feed input displays a text field with the maximum amount of text lines it can contain. Once the text field has reached the maximum height, a scrollbar appears to access the overflowing text.
