 Combobox Mobile Mode To use Combobox in mobile mode:
  • import ComboboxMobileModule
  • Set [mobile] property to true
  • Set [mobileConfig] with MobileModeConfig configuration object to customize the look of the control
Combobox Mobile Mode can be configured globally by providing MOBILE_MODE_CONFIG token. Check Global Configuration page for more information.
