Info Label

Info labels platform component is primarily used to provide information on an object. Colors, text, icons and numbers can be used to easily highlight specific information.

- Info label is itself a component unlike Directive in core.

- Info Label platform component supports the MetaUI (component bases web application).

- Info Label offers additional option like Aria attributes for Accessibility.

- Different color options are available to define the state of the information.

- This component is a small non-interactive numeric or text-based control.
