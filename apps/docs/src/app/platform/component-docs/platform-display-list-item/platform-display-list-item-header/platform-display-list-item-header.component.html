Display List Item

The display list item is the simplest list item. It shows content in the form of labels and text. This component makes use of list-item directive present in fundamental-ngx/core and has additional enhancements on top of it.

Platform specific enhancements:

  • Display-List-Item is implemented as Angular component to hide complexity of dealing with HTML in order to deliver high-order component.
  • Alignment of sub elements title, secondary, footer, group header, navigation are abstracted.
  • Enhanced support for accessibility which covers access by keyboard, mouse and touch.
  • Enhanced compatibility with assistive technology (screen readers).
  • Supports both imperative and declarative way of accepting the items for all the list variants.
