Approval Flow
The Approval Flow component is a complex component which tracks and manages the approval process for a business object. From the Approval Flow component a user is able to:
  • See the progress of approvals
  • Add, edit and delete approvers
  • Add, delete and move approvals
  • Add and remove watchers
  • Send reminder notifications to approvers
The only limitation, for now, it's that the process cannot have multiple final nodes (there should be only one).
