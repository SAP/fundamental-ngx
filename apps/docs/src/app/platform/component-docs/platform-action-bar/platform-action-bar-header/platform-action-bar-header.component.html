Action Bar

Platform ActionBar component is built on top of fundamental-ngx/core

fundamental-ngx/core ActionBarComponent is implemented as set of @Directives to give developers more freedom to assemble the component.

In Platform it is implemented as a @Component hiding the internal complexity from application developer. It abstracts out everything making it easier to use. Full list of enhancements is described below.

Platform specific enhancements:

  • Implemented as Angular component to hide complexity of dealing with HTML in order to delivered high-order component.
  • Optimised number of bindings (@Inputs(), @Outputs) suited more for application developer.
  • Attributes like title, description are parameterised.
  • Introducing 'back button' is made simpler by just setting the showbackbutton flag to true or false.
  • ActionButtonGroup is used to associate actions with action bar component. Individual action buttons or an action menu can be associated.
