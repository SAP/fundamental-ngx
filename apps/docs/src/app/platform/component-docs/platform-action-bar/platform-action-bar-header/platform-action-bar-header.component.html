Action Bar

The Action Bar is located at the top of the page and should contain the page title and main actions for the page. The Platform ActionBar component is built on top of fundamental-ngx/core.

fundamental-ngx/core ActionBarComponent is implemented as a set of @Directives that gives developers more freedom to assemble the component.

The Platform implementation hides the internal complexity from the developer making it easier to use. The full list of enhancements is described below.

Platform specific enhancements:

  • Optimised number of bindings (@Inputs(), @Outputs)
  • Action Bar title and description are presented as attributes to the component
  • Simplified 'back button' implementation
