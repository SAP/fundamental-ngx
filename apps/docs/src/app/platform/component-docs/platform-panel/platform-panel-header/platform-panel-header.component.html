Panel
This is a wrapper of the Platform component present in @fundamental-ngx/core.
It provides component signature for Panel Title and Panel Content directives. Also it includes option to add buttons to panel header (future feature enhancement).

Platform specific enhancements:

  • Develop the Panel and sub-elements as components as opposed to core directive approach.
  • Internalize "expand/collapse" button and controller logic.
  • Include option to add buttons to panel header (future feature enhancement).
