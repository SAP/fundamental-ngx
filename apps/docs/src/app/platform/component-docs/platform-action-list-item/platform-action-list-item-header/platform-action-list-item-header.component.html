Action List Item

The action list item lets the user trigger actions directly from a list item This component makes use of list-item directive present in fundamental-ngx/core and has additional enhancements on top of it.

Platform specific enhancements:

  • Action-List-item is implemented as Angular component to hide complexity of dealing with HTML in order to deliver high-order component.
  • Alignment of sub element title is abstracted.
  • Enhanced support for accessibility which covers access by keyboard, mouse and touch.
  • Enhanced compatibility with screen readers.
  • Supports both imperative and declarative way of accepting the items for all the list variants.
