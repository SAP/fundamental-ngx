Wizard generator
This is a wrapper of the Platform component present in @fundamental-ngx/core.
It provides ability to dynamically generate Wizard without any user-provided templates.

Platform specific enhancements:

  • Steps are dynamically rendered based on steps object input.
  • Branching logic is based on the steps object input.

Developers can also set default form field parameters by calling withConfig() method of the PlatformWizardGeneratorModule during it's import.

For more information, please see Form generator documentation.
