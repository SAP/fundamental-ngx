Wizard generator
This is a wrapper of the Platform component present in @fundamental-ngx/core.
It provides ability to dynamically generate Wizard without any user-provided templates.

Platform specific enhancements:

  • Steps are dynamically rendered based on steps object input.
  • Branching logic is based on the steps object input.

Developers can also set default form field parameters by calling withConfig() method of the PlatformWizardGeneratorModule during it's import.

For more information on avaliable elements, please see: Form generator documentation.

Wizard Generator Elements:

  • input: specifies an input field where the user can enter data.
  • number: specifies an input field where the user can enter a number.
  • password: specifies an input field where the user can enter a password.
  • checkbox: specifies square boxes which are ticked (checked) when the user select one or more options.
  • radio: allows the user to see all options and select one.
  • confirm: is a button which allows to gather all the data from the wizard generator.
  • editor: is a text-field area where the user can include a short text.
  • select: is a combination of select and option components where the list of option is shown in dropdown format.
  • datepicker: is a combination of "input-group", "popover" and "calendar" to select a date.
  • switch: is used to activate or deactivate an element.
  • multi-input: is an input with multiple selection enabled.
