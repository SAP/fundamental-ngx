Menu The Menu can be invoked from any element, by using the fdpMenuTriggerFor directive. Menu with Horizontal Positioning The horizontal position of the menu can be controlled via the xPosition property, which can be set to either 'after' or 'before'. The default value is 'after'. Cascading Menu Menus can be cascaded by adding the fdpMenuTriggerFor directives to MenuItems. Menu Within Scrolling Container

Menus are designed to reposition themselves upon scroll of the page or within a scrollable container. However, In order to accomplish this the Menu/MenuTrigger system is reliant on the CDK Scrolling package

For containers that are scrollable (e.g. via the CSS overflow: scroll property), the user needs to add the cdk-scrollable (or cdkScrollable) directive to that container.
