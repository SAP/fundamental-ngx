Menu The Menu can be invoked from any element, by using the fdpMenuTriggerFor directive. Menu with icons Menu items can be extended with additional content using the following directives from @fundamental-ngx/core:
  • fd-menu-addon - adds icon to menu item. If you want to add the icon before the menu item title, remember to add the position="before"
  • fd-menu-shortcut - adds information about useful shortcuts
 Menu with Horizontal Positioning The horizontal position of the menu can be controlled via the xPosition property, which can be set to either 'after' or 'before'. The default value is 'after'. Cascading Menu Menus can be cascaded by adding the fdpMenuTriggerFor directives to MenuItems. Menu Within Scrolling Container

Menus are designed to reposition themselves upon scroll of the page or within a scrollable container. However, In order to accomplish this the Menu/MenuTrigger system is reliant on the CDK Scrolling package

For containers that are scrollable (e.g. via the CSS overflow: scroll property), the user needs to add the cdkScrollable directive to that container, and import the CDK ScrollingModule.
