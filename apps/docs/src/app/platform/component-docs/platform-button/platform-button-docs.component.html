Button Types The default button type is "action", no modifier is required. Use [type]="'standard'", [type]="'positive'", [type]="'medium'", or [type]="'negative'" to modify the type of the button. Button Sizes Use [compact]="true" for the mobile button size. Button with Icons Use [glyph]="'{{ '{' }}icon-name}'" to add an icon to the button. Button State [disabled]="true" makes the button disabled, when we make it false, it won't be disabled any more, by default disabled is set to false.
