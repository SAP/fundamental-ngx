Button Types The default button does not require modifier. Use buttonType="emphasized", buttonType="host", buttonType="positive",buttonType="attention", buttonType="transparent" or buttonType="negative" to modify the type of the button. Use label="text" to give label to the button. Button Sizes Button support two sized: compact and cozy. Use contentDensity="compact" to set the values.Use label="text" to give label to the button. Button with Icons Use glyph="icon-name" to add an icon to the button and use label="text" to give label to the button.

glyphPosition="before" before and after are used to place the icons beside label. Button State 'disabled' and 'selected' states can be achieved through 'disabled' / 'ariaDisabled' and 'ariaSelected' attributes.Use label="text" to give label to the button.
