Button Types The button component follows fundamental-styles for button guidelines. The default button type is "Default button", no modifier is required. Use the [type]="" property to set the button type. Button Sizes Button support two sized: compact and cozy. Use [contentDensity]="'compact'" to set the values. Button with Icons Use [glyph]="'{{ '{' }}icon-name}'" to add an icon to the button. Button State 'disabled' and 'selected' states can be achieved through 'disabled' / 'ariaDisabled' and 'ariaSelected' attributes.
