Button Types The default button does not require modifier. Use [type]="'standard'", [type]="'positive'", [type]="'medium'", or [type]="'negative'" to modify the type of the button. Button Sizes Button support two sized: compact and cozy Button with Icons Use [glyph]="'{{ '{' }}icon-name}'" to add an icon to the button. Button State [disabled]="true" sets the button disabled
