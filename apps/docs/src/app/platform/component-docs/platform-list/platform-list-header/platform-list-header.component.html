List
List in Platform is built on top of List component present in list-item.

In fundamental-ngx/core, features like title, icon, secondary data, footer, group header are exposed as directives which provides more freedom to developers to assemble the List component. In Platform it is implemented as item components hiding the internal complexity from application developer. It abstracts out the assembling part making it easier to use.

Platform specific enhancements:

  • Implemented as Angular Component to hide complexity of dealing with HTML in order to delivered high-order component.
  • Optimised number of Bindings (@Inputs, @Outputs) suited more for application developer
  • Introducing List with configurable features like Select,Multi-select, Navigations, Deletion, Attachment.
  • Enhanced support for accessibility which covers access by keyboard, mouse and touch. Enhanced compatibility with screen readers.
  • Supports both imperative and declarative way of accepting the items for all the list variants.

  • Serves as a container to create following List item variants :

    • Standard List item
    • Display List item
    • Action List item
    • Feed List item
    • Input List item
  • List container inherits the above mentioned features to all the list item variants
