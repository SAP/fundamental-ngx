List - Virtual scroll We will take help of < cdk-virtual-scroll-viewport>>of **scrollingModule** from angular CDK by setting textSize will help us to achive virtual scrolling. Loading hundreds of elements can be slow in any browser; virtual scrolling enables a performant way to simulate all items being rendered by making the height of the container element the same as the height of total number of elements to be rendered, and then only rendering the items in view.
