Menu Button
The menu button is used for triggering the display of a menu.
In ngx-core, it is an extenstion of ngx-button directive, but in Platform this is a separate component and it reduces the complexity to create Menu Button.

Platform specific enhancements:

  • Menu Button wrapped up as a component.
  • This component is used in conjunction with the Platform Menu Trigger.
  • Menu management is made easy by abstracting the menu definition and linking it with Menu Trigger.
