Basic Dynamic Page Dynamic Page that snaps on scrolling Other than the expand/collapse/pin buttons and Snap on Click feature, scrolling the content also snaps the header. Scrolling down will hide the header, and scrolling back up to the top will bring back the header into view. Tabbed Dynamic Page Specify the [tabLabel] option to internally set up tabbed content for the page. Use (tabChange) property to do something on tab change. To programmatically switch/set the dynamic page tab, call setSelectedTab(id) on the DynamicPageComponent and pass in the id of the tab to switch/set to. Page Responsive Behavior The left and right paddings of the Dynamic Page can be adjusted according to the width of the container with the [size] property. Available values are: small, medium, large and extra-large Disable Header Collapse The collapsing header can be disabled by setting [collapsible] and [pinnable] properties to false. Tabbed Dynamic Page with a Flexible Column Layout Please note to give position: relative to the container holding the dynamic page, so that the page is contained within the column(or any other container) correctly.
