Basic Dynamic Page Dynamic Page that snaps on scrolling Other than the expand/collapse/pin buttons and Snap on Click feature, scrolling the content also snaps the header. Scrolling down will hide the header, and scrolling back up to the top will bring back the header into view. Tabbed Dynamic Page Specify the [tabLabel] option to internally set up tabbed content for the page. Page Responsive Behavior The left and right paddings of the Dynamic Page can be adjusted according to the width of the container with the [size] property. Available values are: small, medium, large and extra-large
