Split Menu Button
Split Menu Button features a button with a label and an expand icon. The label specifies the default action and the expand icon opens a menu that list the available actions. The split menu button provides a quick and compact way for users to initiate a default action, or choose another action.

Platform specific enhancements:

  • Separated creation of Menu reference and Split menu button.
  • Integration of Menu with use of fdpMenuTriggerFor directive.
    (More information on fdpMenuTriggerFor directive available in Platform Menu.)
