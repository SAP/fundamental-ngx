Split Menu Button Behaviors When an option from Menu is selected, it will update the primary button label.If the selected option label is longer in length than the original label for primary button, then the label truncates.This can be controlled by "fixedWidth" input variable. Split Menu Button Types Use type="standard" | "emphasized" | "ghost" | "positive" | "negative" | "attention" | "transparent". you can leave type="" empty to use Standard button as well.
