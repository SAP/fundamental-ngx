The column attribute specify the respective layout for XL size, the rest automatically wrap according to layout pattern. Form Field Group Wrap fields with the [form-field-group] component to group them. You can use [column] and [rank] attribute Change of column number for a field To change the column number in form-group-container for a field while screen size changes, property [columnLayout] can be used. it takes object in the form of XL: 3, L: 2, M: 1, S: 1. Here it shows that on L size screen, field should be dispalyed in 2nd column and when scren size changes to XL size then the field will be displayed in 3rd Column(defaults to L layout value when value for XL not provided).

In this example, we have used custom break-points for screen sizes. XL screen size >= 1540px, L screen size is between 1224px and 1540px, M screen size is between 800px and 1224px, S screen size is less than 800px. Change of isInline value for a field To change the isInline value in form-group-container for a field while screen size changes, property [isInlineLayout] can be used. it takes object in the form of XL: true, L: false, M: true, S: false. Here it shows that on L size screen, field should be dispalyed in vertical mode and when scren size changes to XL size then the field will be displayed inline(defaults to L layout value when value for XL not provided).

In this example, we have used custom break-points for screen sizes. XL screen size >= 1540px, L screen size is between 1224px and 1540px, M screen size is between 800px and 1224px, S screen size is less than 800px.
