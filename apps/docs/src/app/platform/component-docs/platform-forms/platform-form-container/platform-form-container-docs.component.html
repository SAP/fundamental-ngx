The column attribute specify the respective layout for XL size, the rest automatically wrap according to layout pattern. Form Field Group Wrap fields with the [form-field-group] component to group them. You can use [column] and [rank] attribute Change of column number for a field To change the column number in form-group-container for a field while screen size changes, property [columnLayout] can be used. it takes object in the form of XL: 3, L: 2, M: 1. Here it shows that on XL size screen, field should be dispalyed in 3rd column and when scren size to L size then the field will be displayed in 2nd Column. Change of isInline value for a field To change the isInline value in form-group-container for a field while screen size changes, property [isInlineLayout] can be used. it takes object in the form of XL: true, L: false, M: true, S: false. Here it shows that on XL size screen, field should be dispalyed in inLine mode and when scren size to L size then the field will be displayed in vertical.
