Select Modes Following examples presents different modes of the Select control. Use contentDensity="compact" to explicitly set compact mode. Select with Two Columns Select with two columns of information. Note that a secondaryKey and ShowSecondaryText is required. firstColumnRatio and SecondColoumnRatio will help to set required width ratio for each columns by default ratio is 3:2 which is equal to 60%:40%. Select Semantic States Use semantic states to visually highlight current state of the Select. Use contentDensity="compact" to explicitly set compact mode. Custom Control Content With AutoResize The content of the Select control can be customized using a custom template which needs to be referenced in the [controlTemplate] input. Select provides the following context values to the template:
  • Text value of selected option - implicit
  • Reference to selected option - selected
Selected option as a context to custom template. Setting [AutoResize]=true with no specific width helps to resize the select control based on option selected. Select In Mobile Mode To use Select in mobile mode:
  • import SelectMobileModule
  • Set [mobile] property to true
  • Set [mobileConfig] with MobileModeConfig configuration object to customize the look of the control
Select Mobile Mode can be configured globally by providing MOBILE_MODE_CONFIG token. Check Global Configuration page for more information. Select Max Height The Select component allows you to define the [maxHeight]. It can be useful when there are a lot of options given that the select body is be scrollable. By default the maximum height of the Select body is 45% of the browser's inner window height. No Value Select The Select component with[noValueLabel] helps user to move to no value selected state. Do not Wrap the Options We use noWrapText= true by which long text in options won't get wrapped on overflow. They get truncated with ellipsis. Select In A Reactive Form The Select component can be used inside Angular reactive forms.
