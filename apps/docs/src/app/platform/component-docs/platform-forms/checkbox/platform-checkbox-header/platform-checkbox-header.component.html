Checkbox
Platform Checkbox is wrapper implemention on fundamental-ngx/core/checkbox to work with Platform Form. It also works without form.
Implementation simulates the behaviour in primefaces implementation.

Platform specific enhancements:

  • Platform checkbox takes checked state value as a property. so when checkbox is in checked state, control value will be set to value of checkbox. on unchecked state control value will not be defined.
  • Platform Checkbox has capability for binary checkbox. and it can be used outside form.
  • Platform Checkbox has capability for tristate checkbox.
