Checkbox
With checkboxes, all options are visible and the user can make one or more selections. Checkbox is a Platform form component. It can be used without form as well.
  • Checkbox is a HTML element which can be used with form as well as without form.
  • Platform checkbox has only one value, for checked state attribute. It does not take second value for unchecked state attribute.
  • Platform checkbox is specially designed to be used with Platform Form
  • Platform Checkbox implementation has following features.
    1. Binary checkbox with Platform form. In this case checkbox does not take any value. The state(checked or unchecked) is identified through boolean 'true' or 'false'.
    2. Checkbox without Platform form. In this case checkbox can either take a value for selected state. It can also behave like a binary check box by not taking the value but allowing its state to be identified through boolean 'true' or 'false'.
    3. Multiselect checkbox with value with Platform form.
    4. Tristate checkbox with value
    5. Tristate checkbox without value
    6. Compatible with Platform Form for validation and error management
**Required checkbox: Use requiredTrue as Validaltor.
