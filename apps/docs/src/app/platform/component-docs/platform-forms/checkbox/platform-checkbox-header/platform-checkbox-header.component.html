Checkbox
  • Checkbox is a HTML element which can be used with form as well as without form.
  • With checkboxes, all options are visible and the user can make one or more selections.
  • Unlike checkbox in core implementation, platform checkbox expects value of checked state only.
  • Platform checkbox can be used with Platform Form
  • Platform checkbox supports internationalisation with i18n directive
  • Platform Checkbox implementation has following features.
    1. Binary checkbox with form and without form
    2. Binary checkbox without form
    3. Binary checkbox with value without form
    4. Multiple checkbox selection
    5. Tristate checkbox with value and without value
    6. Tristate checkbox without value
    7. Compatible with Platform Form for validation and error management
**Required checkbox: Use requiredTrue as Validaltor.
