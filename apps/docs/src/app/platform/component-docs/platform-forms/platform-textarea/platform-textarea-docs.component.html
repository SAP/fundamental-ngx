Textarea This example shows basic Textarea Textarea with growing option This example shows how Textarea behaves when growing is set.

growingMaxLines - the number of visible lines. Height of textarea is according to this number.
height - if specified, overrides growingMaxLines to set the height of textarea.
cols - approximately, the number of characters in a single line when the width of textarea is not set.
growing - when set to true, resizes the textarea to either growingMaxLines or height, whichever is applied. As user types, the textarea overflows beyond the width to the next line and grows in height until the number of growingMaxLines is reached, or until the height is reached; whichever takes precedence.
