Textarea This example shows basic Textarea Textarea with growing option This example shows how Textarea behaves when growing is set.

growingMaxLines - the number of visible lines. Height of textarea is according to this number.
height - if specified, overrides growingMaxLines to set the height of textarea.
cols - approximately, the number of characters in a single line when the width of textarea is not set.
growing - when set to true, resizes the textarea to either growingMaxLines or height, whichever is applied. As user types, the textarea overflows beyond the width to the next line and grows in height until the number of growingMaxLines is reached, or until the height is reached; whichever takes precedence. This example shows how Textarea works with i18n markers to achieve internationalization support. Translation of the counter message is available directly from the library and will be added to your application's .xlf files on using the following commands from version 0.21.0 onwards:
ng add @fundamental-ngx/platform - if you don't already have ngx/platform installed, use this; or
ng update @fundamental-ngx/platform --migrate-only --from=0.20.0 - if you have ngx/platform installed and want to get the library translations, use this.
You can modify the counter message translation by looking for the i18n trans-unit id platformI18nTextareaCounterMessage in your application's .xlf files.
