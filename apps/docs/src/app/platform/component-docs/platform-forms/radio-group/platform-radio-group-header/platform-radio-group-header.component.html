Radio Button Group
Radio button group is a component specially designed to work with platform form, to create Platform Radio Buttons. Radio button group creates multiple radio buttons and manages it as a single element. There is no need to explicitly associate a group name for every radio button.
Only one radio button can be selected at a time.

Platform specific enhancements:

  • Radio button group is designed to work with platform form. It always requires an associated platform form.
  • Provides default grouping of radio buttons when created using Radio Button Group.
  • Radio buttons can be created from the list of developer provided values (array of strings or list of SelectItem objects).
  • Optionally, if internationalisation is required, radio button group can be created by passing individual radio button elements and placing i18n directive to the radio button element's labels.
  • Radio buttons can be placed in the required layout (default or inline).
