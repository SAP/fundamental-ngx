Radio Button Group
Platform Radio Button group creates multiple radios by using list of values or by passing individual radio buttons. @fundamental-ngx/core does not have an implementation of radio group, instead requires user to build group using individual radio components. Platform Radio Button Group handles all these radios together and selection of only one radio is possible for a form field. It is implemented to work with Platform Form.


Platform specific enhancements:

  • Radio group can create multiple radios by using list of string values
  • Multiple radios can be created using list of SelectItem Objects
  • Radio group can be created by passing radios under radio group
  • It is implemented to work with Platform Form
  • Radio group creates Platform Radio Buttons
  • Radio group can be created using list of Plain Objects (Entities) and specifying lookupKey and displayKey
