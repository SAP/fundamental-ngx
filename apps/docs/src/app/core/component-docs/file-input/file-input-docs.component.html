Simple File Input Used in this context, it will only allow a set of extensions to be added. Invalid File Input An event is fired when invalid files are added through the drag and drop feature. These files can be retrieved and displayed to the user at will. Max File Size It is possible to control the size of the file through the input maxFileSize. The number is in bytes. This feature will keep casual users from entering larger files, but someone knowledgeable enough could potentially bypass this.

NOTE: It is possible to bypass this client-side limit so a server-side check is still necessary.
