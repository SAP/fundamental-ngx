Button Types The default button type is "action", no modifier is required. Use [fdType]="'standard'" [fdType]="'standard'" [fdType]="'emphasized'" [fdType]="'ghost'" [fdType]="'positive'" [fdType]="'negative'" [fdType]="'attention'" [fdType]="'transparent'". Leave [fdType]="" empty to use Action button.

A MenuItemComponent can be passed to the [selected] input to select a menu item. If no [selected] menu option is provided, a title for the button can be provided with [menuActionTitle]. By default, the main action button will keep its original width on load, regardless of the length of the text of the selected option. To have the button width change with the text length, set [fixedWidth]="false". Buttons with Icons Use [glyph]="icon-name" to change an icon of dropdown menu. Programmatic Selection and Open / Close It is possible to control the open state of the popover using MenuComponent.open() and MenuComponent.close() methods. A menu option can also be selected programmatically using the [selected] input. Custom Template for Action Button It is possible to add a custom template to the action button through the fd-split-button-action-title directive.
