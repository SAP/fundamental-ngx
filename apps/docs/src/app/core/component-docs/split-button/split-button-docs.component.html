Button Types The default button type is "action", no modifier is required. Use [fdType]="'standard'" [fdType]="'standard'" [fdType]="'emphasized'" [fdType]="'ghost'" [fdType]="'positive'" [fdType]="'negative'" [fdType]="'attention'" [fdType]="'transparent'". Leave [fdType]="" empty to use Action button Buttons with Icons Use [glyph]="icon-name" to change an icon of dropdown menu. Open/Close Programmatically It is possible to control the open state of the popover using MenuComponent.open() and MenuComponent.close() methods. Custom Template for Action Button It is possible to add a custom template to the action button through the fd-split-button-action-title directive.
