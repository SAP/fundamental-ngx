Theming

Theming module allows developers to easily manipulate appearance of their application.

It has couple of noticeable improvements comparing to the deprecated Theme Switcher API:

  • Automatic theme change without the need of adding custom link elements on the page;
  • Flexible configuration of user custom themes;
  • Ability to change theme fonts "on the fly"

The API requires additional assets from the npm packages: @sap-theming/theming-base-content and fundamental-styles
