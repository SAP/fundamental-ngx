Pagination
Use pagination components in conjunction with Lists, Tables, and Trees to allow for more efficient performance and quick access to specific records in the overall data set.

Pagination links are navigable via the tab key and selected with the space bar or enter key.

If <= 9 pages present - all pages shown, button in active state shown for the current page, otherwise "..." (more) elements shown, input with the current page value shown, can be used to navigate to certain page.

Pres Enter or Space after filling the input with page number to navigate to the certain page.

The pagination component's currentPage value must be updated from your component for the component to work properly.

Pages can be selected programmatically by calling the goToPage(pageNumber) method of the pagination component.
