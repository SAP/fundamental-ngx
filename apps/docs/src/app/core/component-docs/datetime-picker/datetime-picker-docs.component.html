Simple Datetime Picker The simplest form of the datetime picker. It makes use of ngModel to bind to the desired value. The alternative is to use the date input combined with the dateChange output. There is also added [placement] attribute which allows us to decide where popup should be shown The value for the input only changes the model on blur or an enter/return keystroke. Some default options that belong to the time component such as [displaySeconds], [displayMinutes], [displayHours] and [meridian] are based on the DateTimeFormats.display.dateTimeInput options.
For example if Datetime picker display format option contains hours, minutes and day period info then default options will be the next: [displaySeconds]="false", [displayMinutes]="true", [displayHours]="true" and [meridian]="true". The listed above options are applied to the time component only (used internally) and can be defined explicitly if needed. Programmatic Change It is possible to programmatically change the value of the date of the datetime picker and still conserve full functionality. Null Validity Null input values are considered valid by default. Use [allowNull]="false" to make them invalid. Formatting Providing a custom format for the dates is possible through providing DATE_TIME_FORMATS config. Note that when providing a custom format, the [placeholder] input should be set to match the new format as well. The dateTimeFormat pipe can also be utilized to convert the date into a string according to the provided format. Disabled State Date Time Picker in Reactive Form The date-time-picker component may also be used within Angular Reactive Forms. DateTime Picker follows Date Picker with [disableFunction] i18n example. IMPORTANT
The current examples are based on FdDateTimAdapter.
The FdDateTimAdapter is based on native date api (formatting is based on Intl object , parsing is based on Date.parse() ).
Since this adapter has some disadvantages, formatting may vary among browsers and parsing is limited to en locale only.
You can use a custom fundamental adapter based on a third party library: https://momentjs.com/.
Please check examples and documentation on: Moment date time adapter. To change a locale option in DatePicker component, it is needed to set a locale via DatetimeAdapter.setLocale method.
