Datetime Picker

The datetime picker component is an opinionated composition of the fd-popover, fd-calendar and fd-time components to accomplish the UI pattern for picking a date and time.

The datetime picker supports the same keyboard navigability as the calendar and time components. The popover can be opened by focusing the datetime picker input field and closed by focusing the icon button followed by pressing enter or space.

Fully compatible with Angular forms.

IMPORTANT
The Date Time Picker component relies on provided datetime implementation (DatetimeAdapter) and datetime formats (DateTimeFormats).
The current examples are based on FdDatetimeAdapter.
The FdDatetimeAdapter is based on native date api (formatting is based on Intl object , parsing is based on Date.parse() ).
Since this adapter has some disadvantages, formatting may vary among browsers and parsing is limited to en locale only.
You can use a custom fundamental adapter based on a third party library: https://momentjs.com/.
Please check examples and documentation on: Moment date time adapter .
