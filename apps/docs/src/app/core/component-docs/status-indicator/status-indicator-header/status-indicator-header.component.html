Status Indicator

The status indicator displays a SVG images that indicate a pictorial representation of an object. It allows users to fill the content on a numeric scale, fillPercentage typically from 0 (lowest) to 100 (highest) as percentage. If any negative value is passed the default lowest(0) value is applied. The Status Indicator uses filling color representation like status and option available are positive,critical,negative.

The Status Indicator uses size representation like size and option available are:

  • Extra Small (xs)(1.375rem)
  • Medium default (md)(1.875rem)
  • Large (lg) (3.4375rem)
  • Extra Large (xl) (6.25rem)

If you omit these property the default value will be applied.

Note: Two mandatory property are required in order to achieve the image rendering. These properties are derived from the svg html representing an image.

Attribute is array of string path represent the svg html path tag with d attribute form a array of string and pass it to the status indicator as a path attribute.

Attribute is a string viewBox represent the svg html viewBox value of the svg provided.
