Truncate The fd-truncate directive can be used to truncate an element's text by either a pixel value provided to the [fdTruncateWidth] input or a character value provided to the [fdTruncateChars] input.
First, the [fdTruncateState] input must be set to true as the default is false.
Please note that when truncating text by character count, the truncated element can only contain basic text. When truncating by pixels, the element can contain additional items such as icons in the examples below.
Also note that, when using [fdTruncateChars], the inner text cannot be altered. If the inner text may change, please use [fdTruncateWidth].
