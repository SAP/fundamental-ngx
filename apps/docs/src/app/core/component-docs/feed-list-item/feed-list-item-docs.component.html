Sample example

Fields for fd-feed-list-item component are:

  • [authorTitle] is author title for feed list item
  • [text] is text for feed list item

Additional fields

  • Set [isRichText] property to true to activate formatted text. If you enable formatted text, more/less button has disappeared.
  • [authorLink] is url to the author page.
  • [moreLabel] is label for more button.
  • [lessLabel] is label for less button.
  • [maxRows] is count of rows for expand and collapse feature

By default all items in list will display as group in one column and have border-top as a separator and you can hide border-top

  • Set [borderLess] property to true on fd-feed-list to hide border between items
 Avatar examples

Use fd-feed-list-avatar to add avatar into feed-list-item

You can use avatar component or your custom, for example, svg code or any structure with styles.

Note: we defined place for avatar with dimensions 3rem width and 3rem height.

 Actions examples Use fd-feed-list-action to add actions control in feed list item. For example, you can use button or menu components. Footer examples

Use fd-feed-list-footer to add footer in feed list item.

In footer you can use any DOM structure or components.

You can use fd-item-footer-byline directive. This directive has predefined font-size, color and also it'll add dot between each directives. Please see first example.
