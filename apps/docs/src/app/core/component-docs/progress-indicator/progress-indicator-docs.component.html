Default Progress Indicator Show the current progress as a percentage value between 0% and 100%. Alternatively, you can show the current progress as text in addition to the bar. In this case, the text is shown on the right of the bar if the progress is 50% or less. In all other cases, the progress is shown right-aligned on the bar itself. By default, the minimum value is 0, the maximum value is 100 and the progress is represented in percentages. This can be customized by providing a value to the [valueText] input as is displayed in the '80 of 100 MB' example. Progress Indicator States The progress indicator can visualize different value states that are represented by various theme-dependent semantic colors. The states are: normal, success, warning, error, and information, provided via the [state] input. Progress Indicator Truncation If the length of the text exceeds the available space in the progress indicator, the text truncates. In this case, clicking the progress indicator displays an information popover with the full text. Progress Indicator Animation Set the [animation] input to true to enable animation when changing the valueNow input. In the examples here, the first progress indicator has its animation input set to true.
