An example using routerLink The breadcrumb component is a type of navigation that indicates the position of a page within the applicationâ€™s page hierarchy. Users can navigate backward by selecting the previous pages in the navigation path.
The breadcrumb is responsive. When the last breadcrumbs item's edge overflows the available area, the earliest breadcrumb will be added to an overflow menu at the start of the breadcrumb trail. By default, these items are displayed in the menu with the most recently added at the top. This can be changed by setting the [reverse] input to [true]. An example using href Responsive Breadcrumbs The breadcrumb will automatically refer to its parent element to know whether to show breadcrumbs or to collapse them into the overflow menu. You can provide an HTMLElement to the [containerElement] input to set your own container element. Note that the responsiveness feature will not function properly if the containerElement or parent element have a set fixed width.
