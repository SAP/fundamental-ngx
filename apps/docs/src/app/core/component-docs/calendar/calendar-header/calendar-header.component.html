Calendar

Commonly used as the contents of a popover when composing “date-picker”, rarely used on its own as a standalone component.

The calendar header buttons are navigable with the tab key. Pressing enter or the space bar when the month or year is focused will change the day selection grid to the month or year selection grid respectively. The user can then tab into the day/month/year selection grid and use the arrow keys to navigate around the grid and make a selection with the space bar or enter key.

IMPORTANT
The Calendar component relies on provided datetime implementation (DatetimeAdapter) and datetime formats (DateTimeFormats).
The current examples are based on FdDatetimeAdapter.
The FdDatetimeAdapter is based on native date api (formatting is based on Intl object , parsing is based on Date.parse() ).
Since this adapter has some disadvantages, formatting may vary among browsers and parsing is limited to en locale only.
You can use a custom fundamental adapter based on a third party library: https://momentjs.com/.
Please check examples and documentation on: Moment date time adapter.
