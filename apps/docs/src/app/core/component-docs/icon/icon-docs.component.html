SAP Icons SAP-icons is the default font of the component. No need to set the font input property. BusinessSuiteInAppSymbols Icons For BusinessSuiteInAppSymbols icon font you need to set the font input property to font="BusinessSuiteInAppSymbols" TNT Icons For SAP-icons-TNT icon font you need to set the font input property to font="SAP-icons-TNT"
