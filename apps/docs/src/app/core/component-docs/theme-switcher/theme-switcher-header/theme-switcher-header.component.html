Theme Switcher API
Theme switcher API allows user to set specific theme programmatically. There is no longer need for each application to implement its own theme switcher code.
At the moment this won't work for Internet Explorer 11. We will enhance this feature in the near future.
The API requires additional assets from the npm packages: @sap-theming/theming-base-content and fundamental-styles.
