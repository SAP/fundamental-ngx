Theme switcher Css files are switched via choosing option. Setting theme via url Theme can be set by adding property in URL providing name of the theme like in the following example. Parameter is configurable, it can be set by calling setThemeByRoute. See example by adding ?customQueryParam=sap_fiori_3_dark to URL. To make it work properly, you need to have imported RouterModule into the project. In case you want to handle theming only once, after page is loaded, use getThemesFromURL method.
