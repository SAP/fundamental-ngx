Default Bar The default mode of the Bar component is Desktop. For Tablet and Mobile use the Cozy mode by adding [cozy]="true" to fd-bar Bar Design Header Bar Design SubHeader Bar Design Header with SubHeader A bar element can take the whole width of the container by setting the [fullWidth] property to true. Bar Design in Page To use the Bar component as a building block of a Page set the [inPage] property to true. For a Home Page use the [inHomePage] property. Page Responsive Behaviour The left and right paddings of the Bar can be adjusted according to the width of the container with the [size] property. Available values are: s, m_l and xl
