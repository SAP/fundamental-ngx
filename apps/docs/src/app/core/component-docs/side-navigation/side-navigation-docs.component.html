Side navigation with one level Side navigation with titles Use this to group navigation. Titles are not clickable. Side navigation with multiple levels Use this when there is more than one level of hierarchy in the left navigation. Each level can be expanded and collapsed. Side navigation with icons Side navigation collapsed with icons The user can identify which level they are on based on the icon displayed as selected when the navigation is collapsed. Note that the suggested use is when there is only one level of navigation as the user can only see one level of navigation when collapsed.
