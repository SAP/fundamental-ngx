Default Busy Indicator Busy Indicator displays loading animation depending on its state. To set Busy Indicator state use [loading] property. Busy Indicator sizes Busy Indicator can be displayed in three different sizes. To change the size use [size] property. Busy Indicator as wrapper Busy Indicator might be used as a wrapper to display elements or sections of the page behind an overflow. Depending of the wrapped element type, use [block] property to manipulate the display property of the Busy Indicator.
