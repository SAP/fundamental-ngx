Busy Indicator
The busy indicator informs the user about an ongoing operation. Try to avoid showing multiple busy indicators at once. Use the busy indicator if:
  • The ongoing operation covers only part of a screen that has multiple controls
  • You need to display additional information or the user needs to be able to cancel the operation
Do not use the busy indicator if:
  • The operation takes less than one second
  • You need to block the screen because the user is not supposed to start another activity. In this case, use the busy dialog
