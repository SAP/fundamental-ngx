Timeline basic example Fd-timeline is a custom component with dynamic nodes and an accessible DOM structure. You can configure the timeline node template as you want using existing wrapper components and directives, or use your own DOM structure.
You can display a node with or without an icon using an input glyph of fd-timeline-node component. You can also configure content collapse/expand in a node using a maxLines input of fd-timeline-node-body component. This function is disabled by default.
