Simple Date Picker You can use two-way data binding to select a date or range of dates on a calendar. Calendars internally utilize an interface called CalendarDay, which contains the property date. To select a date you will create a new object that contains the property date, whose value is a FdDate model object. Then, using two-way binding syntax, set the selectedDay value to your new object. Note that selecting a day (or range of days) in this manner will override any block/disable functions. The default format for the input is MM/DD/YYYY. See the 'Formatting' section for details on how to change this. The value for the input only changes the model on blur or an enter/return keystroke. Range Date Picker Allows the selection of a range of dates. Only the first and last dates are passed back under ngModel. Internationalization It is possible to internationalize both the aria labels and the month names/weekdays through providing a service. Formatting Providing a custom format for the dates is possible through providing a service. Null Validity Null input values are considered valid by default. Use [allowNull]="false" to make them invalid. Date Picker in Reactive Form The date-picker component may also be used within Angular Reactive Forms. Date Picker in Reactive Form with a Range of Dates Position There is also added [position] attribute which allows us to decide where popup should be shown. Disabled State Complex i18n. To change format or locale options in DatePicker component, it is needed to pass some variables to [format] or [locale]. By default FdDate model is formatted by angular built-in datePipe.
