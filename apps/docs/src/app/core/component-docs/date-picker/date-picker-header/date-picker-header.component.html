Date Picker
The date picker component is an opinionated composition of the "input-group", "popover" and "calendar" components to accomplish the UI pattern for picking a date.

The Time Picker component relies on provided datetime implementation (DatetimeAdapter) and datetime formats (DateTimeFormats).
You can view moment date time adapter implementation on this page.

The date picker supports the same keyboard navigability as the calendar. The calendar can be opened by focusing the date picker input field and closed by focusing the calendar icon button and pressing enter or space.
