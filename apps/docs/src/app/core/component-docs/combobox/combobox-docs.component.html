Standard Combobox The standard version of the component that most people will likely use regularly. Custom Filter The input supports custom filters through the filterFn input function. Simply pass a function which accepts an array and a search term, handle filtering logic within this function, and then return the desired filtered array. In this example, we are only populating the list with results where the search term matches the beginning of the result, as opposed to the usual behavior where the list will be populated if the search term appears anywhere in the string. Custom Search Function A custom search function can be provided with the [searchFn] input. This function will be called when the primary combobox button is clicked, or when the user presses the 'enter' key while focused in the combobox input. Combobox Mobile Mode To use Combobox in mobile mode:
  • import ComboboxMobileModule
  • Set [mobile] property to true
  • Set [mobileConfig] with MobileModeConfig configuration object to customize the look of the control
Combobox Mobile Mode can be configured globally by providing MOBILE_MODE_CONFIG token. Check Global Configuration page for more information. Display Object Property The displayFn input can be used to modify how the results are presented within the combobox. In this example we are making the results all upper-cased. Open State Control By subscribing to event (openChange) it is possible to detect the changes of open state in combobox. By passing [closeOnOutsideClick]="false", [openOnKeyboardEvent]="false" and [triggers]="[]", it is possible to fully control how the user will open and close the results list. Observable Async Example The dropdownValues input can accept an Observable with an async pipe, which could be useful for autocomplete. This example also demonstrates how to create a callback function from data returned by the observable. Custom Item Template The way the items are rendered in the popover list can be customized by providing an itemTemplate to the component. The itemTemplate should align with the contents of an fd-list-item directive. Combobox with Two Columns The [itemTemplate] input can be used to create a combobox with two columns of information. Note that a displayFn is required for this example so the combobox knows how to display the selected item. Combobox with Groups Grouping data within a set of values can be achieved by passing a function to the [groupFn] input. This function should accept an object of key/value pairs as the parameter. Each key will be a group name and the corresponding value for that key will be an array of any type. A [displayFn] can be used to specify what to display from that array. Custom Height Example Use the [maxHeight] input to set a maximum height for the results list. Reactive Form The Combobox component may also be used within Angular Reactive Forms. Use the [communicateByObject] boolean input if you are using custom objects for the dropdownValues or if you want the combobox to only trigger onChange when a value is selected, rather than every time the input changes. Disabled Combobox This example shows how to use a a disabled combobox in a reactive form.
