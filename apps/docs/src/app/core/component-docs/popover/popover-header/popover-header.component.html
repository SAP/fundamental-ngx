Popover

The popover is a wrapping component that accepts a “control” as well as a “body".

A control can be anything that you want to trigger the interaction from. The body will be the contents of what you reveal on the page after triggering the popover. Commonly used as the interaction/wrapping component when composing “dropdowns”, “contextual menus”, etc, when paired with the menu component.

The popover body can have optional Header, Subheader and/or Footer. When the Header is used with a Subheader, the Header should set the [hasSubheader] property to "true". Header, Subheader and Footer have compact mode which is available by setting the [compact] property to "true".

As a general rule, it is suggested that one popover be revealed on the page at any given time. Opening one popover should close all others to prevent multiple layers and collisions of several popovers.

Fundamental-ngx popovers use the overlay package provided by Angular CDK.

IMPORTANT If you place Popover inside scrollable container - you need to add cdkScrollable directive into container and import ScrollingModule into module.
