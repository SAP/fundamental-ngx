24-Hour Time Picker This is the default time picker setting. 12-Hour Time Picker Set [meridian]="true" to use a 12-hour clock. Time Picker Without Seconds Use [displaySeconds]=false to turn off seconds for the time picker. Time Picker Without Minutes and Seconds Use [displaySeconds]="false", [displayMinutes]="false" or [displayHours]="false" to turn off seconds, minutes or hours for the time picker. At least one of them should be enabled. Disabled Time Picker Set [disabled]="true" to disable the time picker. Compact Time Picker Set [compact]="true" to use compact time-picker. Null Validity Null input values are considered valid by default. Use [allowNull]="false" to make them invalid. Formatting Providing a custom format for time is possible through providing service Time Picker in Form The time-picker component may also be used together with Angular reactive forms. Time Picker with Locale The time-picker component may also be used together with moment js to use the locale to determine the format.
