Time Picker

The Time Picker component allows the user to easily set a time using either an HTML5 time input or the Fundamental Library for Angular Time component.

The time picker supports the same keyboard navigability as the time component. The calendar can be opened by focusing the time picker input field and closed by focusing the calendar icon button and pressing enter or space.

The Time Picker component uses Time Component internally, for internationalization configuration, check Internationalization section in Time documentation

IMPORTANT
The Time Picker component relies on provided datetime implementation (DatetimeAdapter) and datetime formats (DateTimeFormats).
The current examples are based on FdDateTimAdapter.
The FdDatetimeAdapter is based on native date api (formatting is based on Intl object , parsing is based on Date.parse() ).
Since this adapter has some disadvantages, formatting may vary among browsers and parsing is limited to en locale only.
You can use a custom fundamental adapter based on a third party library: https://momentjs.com/.
Please check examples and documentation on: Moment date time adapter.
