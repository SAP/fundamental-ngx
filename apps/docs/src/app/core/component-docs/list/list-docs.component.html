Simple list To display a simple list, extend your HTML list implementation with the following directives:
  • fd-list
  • fd-list-item
  • fd-list-title
 Navigation list For a standard list with navigation links, add anchor elements with fd-list-link directive. All items should be navigable. Navigation indicator If there is only a subset of list items that are navigable, they should be indicated with a navigation arrow. Do not show arrows if all items are navigable, instead use a standard list with navigation. To add navigation indicators, add a [navigationIndication]="true" property to the fd-list element. Action list The action list is used to trigger an action directly from a list of options. To create an action item, add [action]="true" to the item. The following example features "Show More" button as an action list element. Filter and sort list List does not provide functionalities like filtering and sorting data. To implement those features, we recommend creating and using pipes to perform operations on a collection of list items. The following example presents a list with filtering and sorting features with the support of a Toolbar component. Secondary item on list The list component can be customized by adding additional information in additional columns. List with icons List can display icons on both sides of the list items. Borderless list It is possible to also remove the borders from the list items by passing [noBorder]="true". Interactive list The interactive ([interactive]="true") mode will force list item to handle hover and active states. Complex list List items can be split into different groups with individual headers, accompanied by a footer at the bottom. Selection You can also make the list items selectable by adding specific classes to the list and list items.
  • Add [selection]="true" to list
  • Add fd-checkbox or fd-radio-button inside list item
 Keyboard support By default, list supports keyboard navigation. Keyboard support can be disabled by passing [enableKeyboardSupport]="false" to the list component. To implement custom handling of the list navigation that escapes list boundaries use (focusEscapeList) as features in the following example. Drag and drop To use drag and drop features in your list, import DragAndDropModule and add the fd-dnd-list directive to the list element and fd-dnd-item to each list item. Infinite scroll list It is possible to implement an infinite scroll function using the fdInfiniteScroll directive. It allows you to load more data as the user scrolls down the parent element. It is important to remember that the InfiniteScrollModule is required for this feature.
