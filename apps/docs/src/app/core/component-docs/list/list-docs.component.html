Simple List Navigation For a standard list with navigation links add anchor elements with fd-list-link directive. All items should be navigable. Navigation Indicator If there is only a subset of list items that are navigable, they should be indicated with a navigation arrow. Do not show arrows if all items are navigable, instead use a Standard List with Navigation. To add navigation indicators, add [navigationIndication]="true" property to the fd-list element. Action List To create action item, add [action]="true" to certain item. List with filtering and sorting Functionalities like filtering or sorting the data are not provided inside list component. It can be easily added by user, by putting some pipes. Below there is example how it can be achieved with toolbar. Secondary Item on List The list component can be customized by adding additional information in additional columns. List with Icons It is possible to put icons on both sides. Borderless List It is possible to also remove the borders from the list items by passing [noBorder]="true". Complex List List items can be split into different groups with individual headers, accompanied by a footer at the bottom. Selection You can also make the list items selectable by adding specific classes to the list and list items.
  • Add [selection]="true" to list
  • Add fd-checkbox or fd-radio-button inside list item
 Keyboard Support Keyboard support is built inside the list component, it supports Arrow Down/Up and emits click event, when Enter/Space keys triggered. It can be disabled, by passing [enableKeyboardSupport]="false" to list component. Also methods triggered on list leave can be customised as on example, by listening to (focusEscapeList) event. Drag and Drop List can be used along with DragAndDropModule. To achieve it, user should add fd-dnd-list to list element and fd-dnd-item to every list item. Infinite Scroll List It is possible to implement an infinite scroll function using the fdInfiniteScroll directive. It allows you to load more data as the user scrolls down the parent element. It is important to remember that the InfiniteScrollModule is required for this feature.
