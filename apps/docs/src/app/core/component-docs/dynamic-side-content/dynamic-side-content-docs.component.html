Basic usage Use fd-dynamic-side-content component with fd-dynamic-side-content-side for additional content and fd-dynamic-side-content-main to hold the main content. By default [position] property is not defined ('none') and in this case the side content will be rendered accordingly to the order that it's projected. Positioning Use [position] option to define the position mode for side content. Allowed values are 'left' | 'right' | 'bottom' | 'equalSplit' | 'none'.
Position defined as 'left', 'right' or 'bottom' takes precedence over content projection order.
[position]="equalSplit" enables equal split mode which provides users with a 50:50 view of the main content and side content. Responsiveness Use [size] option to display the dynamic side content on specific screen sizes. Allowed values are 'sm' | 'md' | 'lg' | 'xl'.
Note that according to the design the component looks the same on medium and large screens.
