Simple Multi Input The standard version of the component that most people will use regularly. Width of the input can be controlled simply through the top-level component element sizing. Overflowing tokens can be reviewed by swiping the tokens left and right. Compact Multi Input The compact multi-input is for large screens. The user can review tokens by pressing the right or left arrows on the keyboard. Display Object Property The input supports objects through the displayFn input. The way the object is displayed to the user is fully customizable through providing a function that accepts the object and returns the string to be displayed. The example below shows displaying an object field in full uppercase. Custom Filter The input supports custom filters through the filterFn input function. Simply pass a function which accepts an array and a search term, and then returns the desired filtered array. Observable Async Example The dropdownValues input can accept an Observable with the async pipe, which could be useful for autocomplete. Multi Input in Reactive Form The multi-input component may also be used within an angular reactive form.
