Default switch with variations Set the [compact] input property to true to render switch in compact mode. An optional text for the active and inactive states of the switch can be set by providing values for the [inactiveText] and [activeText] input properties. Disabled State Switch can be in the disabled state, which prevents the user from interacting with it. Two-Way Binding It is possible to two-way bind to the checked property of the fd-switch. Semantic and Compact Mode It is possible to add semantic mode by passing [semantic]="true" and compact mode by passing [compact]="true" to switch component. Switch used within Reactive Form It is possible to use the fd-switch component within Angular Reactive Forms Example Switch
