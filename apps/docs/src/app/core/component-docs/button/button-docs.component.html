Button Types The button component follows fundamental-styles for button guidelines. The default button type is "Default button", no modifier is required. Use the [fdType]="" property to set the button type. Menu Button In order to use menu button, use [fdMenu]="true" Button Sizes Use (or skip) [compact]="false" for the mobile button size. Button with Icons Use [glyph]="'{{ '{' }}icon-name}'" to add an icon to the button. Button State 'disabled' and 'selected' states can be achieved through native 'disabled' / 'aria-disabled' and 'aria-selected' attributes. The inputs for button will be translated into classes attributed to the fundamental-styles guidelines.
For example:
if positive is passed for fdType, the fd-button--positive class will be added to the element.
if compact is passed for size, the class fd-button--compact will be added to the element.
