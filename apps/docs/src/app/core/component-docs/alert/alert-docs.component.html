Alert Types A number of alert types are available. They offer preset styling for different contextual situations. Alert as Overlay The AlertService is available for injection and handles most of the work for this usage. It accepts a component type, a template ref or a string as the alert content. If a component is provided as content, it should be added to the entryComponents of your application. Inline Alert To make use of inputs such as mousePersist and duration in an inline alert, a call to the alert open() method is required. Otherwise, the alert will simply be shown without any complex behaviour.

Note that inline alerts are not removed from the dom. Alert with Customizable Width Alert width can be customized by adding the [width] attribute on the selector.
