Resizable Card Layout Initial width and height can be passed directly to card item. Layout is displayed by sorting cards on rank field. Individual cards can also be made Non-resizable by setting resizable property to false. Resizable Card Layout using Layout config Initial width and height for cards can also be passed from the layout. Array of ResizableCardItemConfig can be passed to layout layoutConfig property. Layout itself assign width and height of card according to value given in ResizableCardItemConfig. Resizable Card Layout using Item config Initial width and height for cards can also be passed to card item in form of ResizableCardItemConfig by setting config property. width and height of card are assigned accordingly.
