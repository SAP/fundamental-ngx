Radio Button
Radio buttons allow the user to see all options and select one. Generally, this is used when there are between 2-3 options. This component can also be disabled and displayed in a row.
Keyboard support for radio button selection is handled via the arrow keys.
On new version of chromium engine (Opera, Microsoft Edge, Chrome) radio button component causes crash.
