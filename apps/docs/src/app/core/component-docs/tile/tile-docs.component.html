Generic Tile Sizes The large (default) tiles are designed for screens larger than 374px. For smaller screens use the small tiles size="s". The generic tile also supports two tile dimensions - 1×1 (default) and 2×1 ([double]="true").
Generic Tile Content Columns The tiles header, content and footer can be split into two columns with a 0.25rem vertical padding. Use the [twoColumn]="true" input to achieve the 2-col design.
Launch Tile Use type="launch" for launch tiles.
KPI Tile Use type="kpi" for KPI tiles.
Action Tile Use the [action] boolean input to place a tile in action mode. This will place an overlay over the tile, and you should also provide fd-tile-action-close and fd-tile-action-indicator button directives to handle the action functionality.
Badge Tile Badges are used to display very short and important information. Use a badge on a Tile if the Tile is new or has been newly added to a page. Tiles do not display badges in action/edit mode.
Feed Tile The Feed tile can be used to show new notification content in a news feed. If there are no new notifications, the tile displays the most recent notification. To create a Feed tile, use the type="feed" input.

Clickable Tile Tile clickable by the default, but you can change it behavior by providing [clickable] input property. When tile is clickable it can get focus & event will be emitted on click, enter and space press, listen for (tileClick).
