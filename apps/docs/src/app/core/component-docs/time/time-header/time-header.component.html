Time

The time component is used for a single time value. Multiple components can be used in the time-picker to assemble a clock time. A max of four will account for hours, minutes, seconds and period of the day. It will be rare to see this component used outside of it being composed in the time-picker component.

Time component buttons are navigable via the tab key and can be pressed via the space bar or enter key.

IMPORTANT
The Time Picker component relies on provided datetime implementation (DatetimeAdapter) and datetime formats (DateTimeFormats).
The current examples are based on FdDatetimeAdapter.
The FdDatetimeAdapter is based on native date api (formatting is based on Intl object , parsing is based on Date.parse() ).
Since this adapter has some disadvantages, formatting may vary among browsers and parsing is limited to en locale only.
You can use a custom fundamental adapter based on a third party library: https://momentjs.com/.
Please check examples and documentation on: Moment date time adapter.
