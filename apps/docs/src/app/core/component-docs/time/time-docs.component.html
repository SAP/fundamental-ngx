24-Hour Clock This is the default clock setting. 12-Hour Clock You can use a Meridian 12-hour clock by setting [meridian]="true". The passed time object will still contain a 24-hour clock. No Spinners For mobile, or tablet devices spinners should be removed from time component. It can be done by passing [spinnerButtons]="false". Sizes You can change size of time component by passing
  • [compact]="true", which is dedicated for desktop devices and makes time items smaller.
  • [tablet]="true", which is dedicated for tablets and changes container spacing.
 Programmatically Changed Time Makes it possible to programmatically change the value of the currently selected time. By programmatically changing the value of this property, the time that is highlighted on the calendar will change accordingly. Clock Without Seconds Use [displaySeconds]="false" to hide the seconds column. Clock Without Minutes and Seconds Use [displaySeconds]="false", [displayMinutes]="false" or [displayHours]="false" to turn off seconds, minutes or hours for the time picker. At least one of them should be enabled. Time With Two Digits Add [keepTwoDigits]="true" to always have two digits in hours, minutes and seconds input. It also can be used in TimePicker and DateTimePicker components. Time in Reactive Form The time component may also be used within a form. The values within the form may then be accessed through the usual methods used for accessing data within Angular reactive forms.
