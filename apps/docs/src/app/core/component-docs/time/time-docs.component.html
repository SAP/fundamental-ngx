24-Hour Clock This is the default clock setting. 12-Hour Clock You can use a Meridian 12-hour clock by setting [meridian]="true". The passed time object will still contain a 24-hour clock. Clock Without Spinners Use [spinners]="false" to hide the spinners. Clock Without Seconds Use [displaySeconds]="false" to hide the seconds input. Clock Without Minutes and Seconds Use [displaySeconds]="false", [displayMinutes]="false" or [displayHours]="false" to turn off seconds, minutes or hours for the time picker. At least one of them should be enabled. Disabled State You can disable the entire component by setting [disabled]="true". Internationalization The time component uses a custom service called TimeI18nLabels to give aria-labels to the buttons and inputs. Inspect the HTML elements to see the Spanish aria-labels. The time component also uses a custom service called TimeI18n to allow the possibility of changing the name of the meridian modifier. In the example below, the german vorm. and nach. are shown. Also it is possible to modify placeholders of the inputs, such as hours, minutes, seconds, and meridian. Besides service allows to define if meridian modifier should be validated case sensitive. Time in Reactive Form The time component may also be used within a form. The values within the form may then be accessed through the usual methods used for accessing data within Angular reactive forms.
