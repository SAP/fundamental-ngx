Invalid File Upload An event is fired when invalid files are added through the drag and drop feature. These files can be retrieved and displayed to the user at will. Min File Size It is possible to control the size of the file through the input minFileSize. The size you can specify in Byte, KB, MB, GB, TB. eg 20Byte, 2KB, 1MB etc... This feature will keep casual users from entering smaller files to reduce network traffic, but someone knowledgeable enough could potentially bypass this.

NOTE: It is possible to bypass this client-side limit so a server-side check is still necessary.
