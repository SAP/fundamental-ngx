Moment Datetime Adapter
DEPRECATED. Since Moment library is now considered as legacy, we are deprecating the MomentDatetimeAdapter in favour of Dayjs. Please use DayjsDatetimeAdapter instead The Date Picker, Time Picker, Date Time Picker and similar components rely on provided datetime implementation (DatetimeAdapter) and datetime formats (DateTimeFormats).

These components could be used with FdDatetimeAdapter, based on the JavaScript's native Date object, but one of the biggest shortcomings of the native Date object is the inability to set the parse format. As an alternative could be the MomentDateAdapter or a custom DateAdapter that works with the formatting/parsing library of your choice.

Please consider that you need to install this package separately.
