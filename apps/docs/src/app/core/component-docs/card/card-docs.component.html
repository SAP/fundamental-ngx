Standard Card Use fd-card component with a fd-card-header and fd-card-content to create a simple card Compact mode Use [compact]="true" option to make the card compact Card loader Use [isLoading]="true" along with fd-card-loader component to show alternative content during data loading Card footer Use fd-card-footer component to put actions in the card footer Analytical Card Use fd-card-kpi-header component along with [cardType]="analytical" to display analytical data in the card header.

Here for illustration we use google charts library to render a chart within the card content.

 Table Card Use [cardType]="table" to display table card Bar Chart List In order to display bar chart list in a card we can use list styles along with a custom bar component. For this example in order to display bar chart we created a simple fd-card-bar component. For card with list it is not recommended to keep header interactive, or navigable. Such a header can be achieved by adding [interactive]="false" to fd-card-header element.
