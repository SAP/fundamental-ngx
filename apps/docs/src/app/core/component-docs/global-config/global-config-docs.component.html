Dialog global config Dialog default configuration can be provided using DIALOG_DEFAULT_CONFIG Injection Token. To globally configure the dialog provide DialogConfig for DIALOG_DEFAULT_CONFIG in your application root module. Message box global config Message box default configuration can be provided using MESSAGE_BOX_DEFAULT_CONFIG Injection Token. To globally configure the message box provide MessageBoxConfig for MESSAGE_BOX_DEFAULT_CONFIG in your application root module. Control Mobile mode global config Control mobile mode default configuration can be provided using MOBILE_MODE_CONFIG Injection Token. To globally configure control mobile mode:
  • Provide MobileModeConfigToken value for MOBILE_MODE_CONFIG token
  • Specify the target control in MobileModeConfigToken.target using MobileModeControl
Popover Mobile mode global config Popover Mobile default configuration can be provided using MOBILE_MODE_CONFIG Injection Token. To globally configure the popover mobile, provide MobileModeConfigToken.target for POPOVER_MOBILE_CONFIG in your application root module.
