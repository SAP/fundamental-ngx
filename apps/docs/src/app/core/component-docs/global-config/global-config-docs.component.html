Dialog global config Dialog default configuration can be provided using DIALOG_DEFAULT_CONFIG Injection Token. To globally configure the Dialog provide DialogConfig for DIALOG_DEFAULT_CONFIG in your application root module. Control Mobile mode global config Control mobile mode default configuration can be provided using MOBILE_MODE_CONFIG Injection Token. To globally configure control mobile mode:
  • Provide MobileModeConfigToken value for MOBILE_MODE_CONFIG token
  • Specify the target control in MobileModeConfigToken.target using MobileModeControl
