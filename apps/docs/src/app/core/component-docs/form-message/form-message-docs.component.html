Form Message Input Group The Form Input Message Group has a built-in popover component which uses most of the popover properties.
At a minimum, it should contain a control element and a FormMessage. The Form Message can be paired with one of the components listed below:
  • Input
  • Textarea
  • Input Group
  • Multi Input
 Form Message State in Input Type The Form Message offers four value states listed below:
  • Information: The message is informative guiding the user on how to fill out the input.
  • Warning: The message is giving the user warnings on the data input format although it doesn't stop the user with moving forward.
  • Error: The message tells the user that the data entered is invalid and the user cannot move forward.
  • Success: The message shows that the user has entered the data correctly.
The value state message can be either a plain text or a formatted text.
