Fixed Card Layout
The fixed card layout supports automatic, easy and fast card design. The cards are ordered in responsive and collapsible columns.The cards are arranged as a “Z” flow: from left to right, starting with the first card on the top left of the page. Notes: we have used angular-resize-event library for detecting layout size change in documentation examples. Example: when we collapse side navigation bar, available size for layout changes.Using this library, we are able to detect the layout size change and then refresh the layout.
