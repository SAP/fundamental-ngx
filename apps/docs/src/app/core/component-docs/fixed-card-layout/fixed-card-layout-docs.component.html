Default Fixed Card Layout Auto arrangement of cards in fixed card layout. The layout is responsive and cards gets re-arranged when window resize happens or some cards gets hidden or shown. Disabling drag and drop Behavior of dragging cards can be disabled and enabled. Mobile display In mobile display as well layout is consistent and on changing orientation, cards will re-distribute and arrange itself. Custom card width

By default, calculations for the amount of the available columns takes into consideration minimum card width, which is by default equals 320px (20rem).

This parameter can be changed by providing [cardMinimumWidth] input property. The value should be a number of pixels.
