Formatted text component
The FormattedText component allows the usage of a limited set of tags for inline display of formatted text in HTML format.
You can safely append formatted the text using HTML tags or embed formatted text.
The following tags are supported:
  • a
  • abbr
  • blockquote
  • br
  • cite
  • code
  • em
  • h1-6
  • p
  • pre
  • strong
  • span
  • u
  • dl
  • dt
  • dd
  • ul
  • ol
  • li
