Basic Dynamic Page There is example with basic DynamicPage, without collapsible subheader. Dynamic Page with Tabs There is used tab component inside DynamicPage - Content should be wrapped in tab-items Dynamic Page inside Column Layout Responsive Dynamic Page AutoResponsive DynamicPage is enabled by default and can be disabled by passing [autoResponsive]="false". This will auto apply certain sizes for breakpoints:
  • max-width: 599px - small
  • min-width: 600px and max-width: 1023px - medium
  • min-width: 1024px and max-width: 1439px - large
  • min-width: 1440px - extra large
 Dynamic Page with Facets The Dynamic Page header can consist of various facet types, such as Form facet, Key-value facet, Image facet, and Rating Indicator facet. When an Image facet is used in the header content, collapsing the header will bring the image facet next to the title. Expanding the header will place it back to its original place.
