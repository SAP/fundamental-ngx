This will auto apply certain sizes for breakpoints:
  • max-width: 599px - small
  • min-width: 600px and max-width: 1023px - medium
  • min-width: 1024px and max-width: 1439px - large
  • min-width: 1440px - extra large
 Dynamic Page with Facets The Dynamic Page header can consist of various facet types, such as Form facet, Key-value facet, Image facet, and Rating Indicator facet. When an Image facet is used in the header content, collapsing the header will bring the image facet next to the title. Expanding the header will place it back to its original place. Dynamic height of parent element

By default, Dynamic Dage component automatically calculates amount of free space it can take, and recalculates it on window resize.

If you plan on using Dynamic Page component inside container with dynamic content height (e.g. some elements inside container can appear after Dynamic Page component has been initiated), you need to apply fd-dynamic-page-wrapper directive to that container in order for Dynamic Page component to listen container's size changes instead of window size changes.
