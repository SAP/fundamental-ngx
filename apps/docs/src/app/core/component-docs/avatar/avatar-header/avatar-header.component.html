Avatar

The Avatar is used for displaying images. These can be user profiles, user initials, placeholder images,icons, product images, etc.
Use the Avatar only in case you need to display an image, initials, placeholder for a user, icons, placeholder images or images with transparent background.
Do not use the Avatar to include an image for any other case.

Size variations:
  • Extra Small (2rem)
  • Small (3rem)
  • Medium (4rem)
  • Large (5rem)
  • Extra Large (7rem)
