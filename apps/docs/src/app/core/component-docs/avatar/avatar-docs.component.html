Icon Use the [glyph] property to specify the name of the icon. The size of the Avatar is controlled by the [size] property. If omitted, the default size of the Avatar is set to large (l). Initials The initials stand for the first name(s) and last name(s) of a person. Which name comes first depends on the language-specific settings. Circle Set [circle]="true" for a circular shape of the Avatar. Background Image To provide an image resource (url or data:base64) use the [image] property.
By default, the size of the image is scaled up to completely cover the control area.
If you want to scale the image as large as possible within its container
without cropping or stretching the image, set [contain]="true" Transparent To remove the background of the Avatar set [transparent]="true". Placeholder Background To apply a Placeholder Background set [placeholder]="true".
Placeholder images are used when no other image is available.
The default placeholder for an avatar is a gender-neutral person icon inside a circle.
For a business image is a neutral product icon inside a square can be used or you can specify your own default placeholder icon. Tile Icon Background For Tile Background set [tile]="true". Accent Colors By default, the placeholder background color is set to blue. Use [colorAccent] property to change the background color. Options include numbers from 1 to 10. For example: [colorAccent]="6". If you want to use random accent color, use [random]="true" property. Zoom Icon To specify the zoom icon use the [zoomGlyph] property. Borders A border can be added with [border]="true".
