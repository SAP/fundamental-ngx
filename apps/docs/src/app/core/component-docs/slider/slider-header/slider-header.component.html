Slider

Slider control represents a numerical range and a handle. The purpose of the control is to enable visual selection of a value in a continuous numerical range by moving an adjustable handle. Slider can also become a Range Slider. A Range Slider is a user interface control that lets the user to select a value range within a predefined numerical interval.

Changing the Value

If the slider is editable, the hand cursor appears when the user hovers over the grip.

The user can change the slider setting in two ways:

  • By using drag and drop to adjust the grip
  • By clicking the bar. The grip then moves to this new position.
  • Enter the desired value in the input field (if available)
  • Keyboard Arrows: UP, DOWN, LEFT, RIGHT or SHIFT + UP, DOWN, LEFT, RIGHT

The grip can be moved with or without increments based on the predefined steps.
