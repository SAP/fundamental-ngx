Micro Process Flow

The Micro Process Flow control can be used to track the progress of a process.

It is best used for linear processes, such as document, order, and approval flows, and can be embedded in tiles, tables, and other containers.

 Independent steps

If you don't want to display connector line between items, provide [independentSteps]="true" for the fd-micro-process-flow component.

 Status indicator icons / other controls / custom icons

Additionaly, developers can use different components as indicators:

  • Status indicator
  • Button
  • Label
  • Link
  • Avatar

 Micro process flow overflow

If combined width of micro process flow is larger than the container width, component will automatically render navigational buttons and set amount of items already shown and how many items left to display.

Developers can also pass custom easing function and provide different transition time
