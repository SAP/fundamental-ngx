Individual Type Focus on Each Person (type="individual").
The avatars are displayed next to each other and don’t overlap. Use this type if each person is as important as the group as a whole, and the user’s main focus is to get more information about some individuals in the group. Group Type Focus on the Sum of People (type="group").
The group type displays overlapping avatars. Use this type if it’s important to display the sum of people at first sight, rather than the individuals.
