Template Directives Use the fdSkeleton template directive to set the skeleton state for the components subtree. Provide your own template using the fdSkeletonTemplate input property of the directive. You could use *fdSkeleton="skeletonState; template: skeletonTemplate" shorthand.

You could change the skeleton state to the 'global' value and it will react only to the global skeleton state changes.

You could also use omit the fdSkeletonTemplate input property of the fdSkeleton directive and use the separate fdSkeletonTemplate template directive severals levels deeper. This separate directive just listen to the skeleton state set somewhere above (via the directive or the service) in the components subtree. Component Bare skeleton component that can be used as a placeholder for content instead of the spinner component. It supports setting width, height, type - may be circle | text | rectangle (default) and disabling animation if needed (enabled by default).

If type set to text then height input property is ignored and height will be automatically calculated.

width input property can be set to 'rand' so the skeleton will be randomly 30-90% in width. It's also set like that by default.
