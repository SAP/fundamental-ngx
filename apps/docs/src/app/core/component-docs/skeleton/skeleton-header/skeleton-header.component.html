Skeleton
Skeleton component to be used as a placeholder for content instead of the spinner component.

To provide SkeletonService that manage global skeleton state you need to use forRoot method of SkeletonModule in the root module of your application. It's also possible to set defaultSkeletonState(true | false (default)) via this way.

Notice: We will iteratively add skeleton state support out of the box to all the components but before it happened please provide your own templates.
