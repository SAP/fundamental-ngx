Carousel with one active item By default fd-carousel displays one active item and others remains hidden. Pagination container appears on bottom with dot indicators with active dot to show active item. Carousel with vertical direction carousel movement is possible in vertical direction as well. Carousel with multiple active item Carousel can display multiple active item at a time as well. On left navigation button click, right most item hides and a new item is shown on left side. On right navigation button click, left most item hides and a new item is shown on right side. visibleSlidesCount property is used to set, how many item will be active at a time. Ex. visibleSlidesCount=3. Carousel with no page indicator Page indicator is optional in carousel. It has option to hide page indicator by setting pageIndicator=false. Carousel with hidden navigation buttons In Mobile display, navigation buttons are not required for carousel. Navigation will be done by swiping the active item, either left side or right side. Carousel with navigation inside content area Navigation buttons can be placed in page indicator as well as inside content area.By default navigation button is placed inside page indicator container. For placing navigation button inside content area use navigatorInPageIndicator=false. Carousel with looped navigation Carousel item navigation can be looped by setting loop=true. By default loop is false and looping is not enabled on carousel. Carousel error message when no item is loaded Error message can be displayed as carousel content when loading item is not successful.
