Navigation Indicator

If only a subset of the list items are navigable you should indicate those by displaying a navigation arrow. This is achieved by applying the [navigationIndicator] input to the ul element. Also you will need to add the [navigationIndicator] and [navigated] inputs to any navigable fd-list-link directives. Do not show arrows if all items are navigable. In this case use a byline list with navigation.
