Simple menu To create simple menu use following directives:
  • fd-menu-item - menu list element
  • fd-menu-interactive - menu interactive element
  • fd-menu-title - menu element title
 Menu with icons Menu items can be extended with additional content using the following directives:
  • fd-menu-addon - adds icon to menu item. If you want to add the icon before the menu item title, remember to add the [position]="'before'"
  • fd-menu-shortcut - adds information about useful shortcuts
 Menu with separator and compact mode To add a visual separation line between menu items, use the fd-menu-separator component.
To display Menu in compact mode use [compact]="true". Menu with submenu Each Menu item can have its submenu. To create a submenu, do the following:
  • create an ng-template containing menu items, using the standard menu item markup
  • attach your template to the parent menu item by passing the TemplateRef to its [subMenu]
If you are using fd-menu-addon to create icon indicating submenu level, remember to add [submenuIndicator]="true". The important thing is to create templates with submenu items inside the fd-menu tag, so Angular correctly attaches them to the DI tree.
