Simple menu To create simple menu compose you menu items using following directives:
  • fd-menu-item - menu list element
  • fd-menu-link - menu interactive element
  • fd-menu-title - menu element title
 Menu with icons Menu item can be extended with additional content using following directives:
  • fd-menu-icon - to add icon to menu item. If you use fd-menu-icon after menu item title remember to add set [position]="'after'"
  • fd-menu-shortcut - to add information about useful shortcuts
 Menu with separator and compact mode To add visual separation line between menu items use fd-menu-separator component.
To display Menu in compact mode use [compact]="true". Menu with submenu Each Menu item can have its submenu. To create submenu:
  • create ng-template containing menu items, using standard menu item markup
  • attach created template to parent menu item by passing TemplateRef to its [subMenu]
The important thing is to create templates with submenu items inside fd-menu tag, so Angular correctly attaches them to DI tree. Menu in mobile mode To open Menu in mobile mode use [mobile]="true" option. In mobile mode the Menu is opened inside Dialog component taking whole viewport of the device. When going through nested levels of the Menu, name of the parent menu item will display in Dialogs header. To add initial header visible in the Dialogs header at the first level on the Menu use [mobileMenuTitle].
