Grid columns

Grid column width can be set using the fdLayoutGridCol property.

Column offset

Space between columns can be added using the fdLayoutGridColOffset property.

Responsive columns

Responsiveness can be achieved by setting a column width for each size using the fdLayoutGridCol fdLayoutGridColM fdLayoutGridColL fdLayoutGridColXl properties.

Grid row

Rows can be defined by adding divs with the fdLayoutGridRow directive.

Without gap

Rows without gaps can be achieved by using the [noGap] input.
