Grid columns Grid column width can be set using the fdLayoutGridCol property. Growing columns Growing column takes all available width in the row. To create a growing column add colGrow attribute to fdLayoutGridCol element. Column offset Space between columns can be added using the fdLayoutGridColOffset property. Responsive columns Responsiveness can be achieved by setting a column width for each size using the fdLayoutGridCol fdLayoutGridColM fdLayoutGridColL fdLayoutGridColXl properties. Responsiveness works with or without offsets.

Responsive Breakpoints

  • Default until next specified breakpoint: fdLayoutGridCol will be used until the next specified minimum breakpoint, column takes x/12 space of the row.
  • Medium 600px: colMd will be used from 600px to the next specified minimum breakpoint, column takes x/12 space of the row.
  • Large 1024px: colLg will be used from 1024px to the next specified minimum breakpoint, column takes x/12 space of the row.
  • Extra-large 1440px: colXl will be used from 1440px and larger, column takes x/12 space of the row.
Responsive offset Offsets can also be used with the responsive layout grid. Grid row Rows can be defined by adding divs with the fdLayoutGridRow directive. Without gap Rows without gaps can be achieved by inputs:
- [noVerticalGap] - to remove vertical gap,
- [noHorizontalGap] - to remove horizontal gap,
- [noGap] - to remove both Nesting columns Add a row in between inner rows when using nesting. It will add a gutter in between rows.
