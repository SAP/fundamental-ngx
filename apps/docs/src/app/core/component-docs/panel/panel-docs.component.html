Expandable Panel Expandable panels are much like fixed panels, except their content can be expanded and collapsed (including the info toolbar, if available). Fixed Panel Fixed panels are useful for grouping custom content. They include headers and info toolbars. To create a fixed panel, add [fixed]="true" to the Panel. Compact Panel For a compact panel, set the compact property to true.
