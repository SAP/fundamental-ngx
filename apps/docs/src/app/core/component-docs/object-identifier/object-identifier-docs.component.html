Title To display the object identifier as a title, add some text inside fd-object-identifier component. Bold To display the object identifier's title in bold, add [bold]="true". Links To display the object identifier as a link, add fd-link component inside fd-object-identifier component. Descriptive To display the object identifier's title with descriptive text, add [description]="'text'". Tables To display the object identifier in a table, add the fd-object-identifier within the table cells, along with [medium]="true", to reduce size of title.
