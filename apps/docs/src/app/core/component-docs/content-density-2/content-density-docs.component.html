ContentDensityControllerService Content Density can be maintained app-wide using the ContentDensityControllerService, and components can subscribe to a Observable managed by the service and use that to handle changes to the content density. Before using it, you should add ContentDensityModule to imports array of AppModule, or other root module in your application.
Note that many components only come in one size, and others (such as the button) only have compact and cozy sizes, so setting the size to condensed will have same effect as compact.
All components that have content density options will first check to see if the developer has provided a fdContentDensity directive, and if not, it will then refer to the value set in the ContentDensityControllerService. Directives Usage of directive
