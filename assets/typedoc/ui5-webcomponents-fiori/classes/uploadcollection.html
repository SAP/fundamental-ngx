Readonly slots

slots: {
    description: string;
    name: string;
}[] = ...

Available slots for content projection in this component.

Slots allow you to insert custom content into predefined areas of the web component. Use the slot attribute on child elements to target specific slots.

  • (default): Defines the items of the ui5-upload-collection.

Note:* Use ui5-upload-collection-item for the intended design.

  • header: Defines the ui5-upload-collection header.

Note:* If header slot is provided, the labelling of the UploadCollection is a responsibility of the application developer. accessibleName should be used.

Type declaration

  • description: string
  • name: string
Example
<ui5-upload-collection>
  <div slot="header">Custom header content</div>
  <p>Default slot content</p>
</ui5-upload-collection>
