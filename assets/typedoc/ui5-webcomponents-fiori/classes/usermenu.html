Readonly slots

slots: {
    description: string;
    name: string;
}[] = ...

Available slots for content projection in this component.

Slots allow you to insert custom content into predefined areas of the web component. Use the slot attribute on child elements to target specific slots.

  • (default): Defines the menu items.
  • accounts: Defines the user accounts.

Note:* If one item is used, it will be shown as the selected one. If more than one item is used, the first one will be shown as selected unless there is an item with selected property set to true.

Type declaration

  • description: string
  • name: string
Example
<ui5-user-menu>
  <div slot="header">Custom header content</div>
  <p>Default slot content</p>
</ui5-user-menu>
