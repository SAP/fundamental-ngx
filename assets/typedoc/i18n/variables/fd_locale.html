FD_LOCALE: InjectionToken<Observable<string>> = ...
Deprecated

Use FD_LOCALE_SIGNAL instead for better performance and zoneless compatibility. Observable-based locale token (legacy). Will be removed in a future version.

Migration:

// Before
locale$ = inject(FD_LOCALE);

// After
localeSignal = inject(FD_LOCALE_SIGNAL);
// Or convert to Observable if needed:
locale$ = toObservable(inject(FD_LOCALE_SIGNAL));
