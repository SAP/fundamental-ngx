• detectLanguage(locale): FdLanguage

  • Maps a locale string (e.g. from Angular LOCALE_ID) to the best-matching built-in FdLanguage. Falls back to FD_LANGUAGE_ENGLISH when no match is found.

    Resolution order:

    1. Exact locale match (case-insensitive)
    2. Chinese region-to-script mapping (zh-CN → zh-Hans)
    3. Locale alias resolution (nb → no)
    4. Base language match (pt-BR → pt)
    5. Fallback to English

    Parameters

    • locale: string

    Returns FdLanguage
