• provideDayjsDatetimeAdapter(): EnvironmentProviders

  • Returns environment providers that register DayjsDatetimeAdapter and the default dayjs date-time formats.

    Use this in bootstrapApplication or route providers instead of importing the module:

    bootstrapApplication(AppComponent, {
        providers: [provideDayjsDatetimeAdapter()]
    });

    Returns EnvironmentProviders
