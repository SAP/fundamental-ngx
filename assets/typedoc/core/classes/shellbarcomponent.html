The shellbar offers consistent, responsive navigation across all products and applications. Includes support for branding, product navigation, search, notifications, and user settings. Shellbar is a composite component comprised of mandatory and optional elements.

sideNav

sideNav: boolean | SideNavigationInterface = false

Whether the Shellbar is used with Side Navigation. When set to true, the responsive paddings are not applied. Can also accept a template variable referring to a SideNavigationInterface (fd-side-nav or fdx-side-nav), and will add the corresponding class.
