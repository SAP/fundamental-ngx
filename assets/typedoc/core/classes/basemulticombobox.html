Abstract displayKey

displayKey: InputSignal<string>

The property name used to display item labels in the dropdown. For primitive types (strings, numbers), this is ignored and the value itself is displayed. For objects, this specifies which property to use as the display text.

Example
displayKey = 'name'; // Displays item.name for each item

Abstract groupKey

groupKey: InputSignal<string>

The property name used to group items when group is enabled. This key is used to extract the grouping value from each item object.

Example
// If items are: [{name: 'Apple', category: 'Fruit'}, {name: 'Carrot', category: 'Vegetable'}]
groupKey = 'category'; // Groups by 'Fruit' and 'Vegetable'
