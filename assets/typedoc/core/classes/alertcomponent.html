Selector: <fd-alert>

deprecated

Alert component is depricated since version 0.16.0 Message Strip component should be used instead.

The component that represents an alert. It can be only be used inline. If the AlertService is used, this component is auto-generated.

dismissLabel

Aria label for the dismiss button.

dismissible

Whether the alert is dismissible.

duration

duration: number = 10000

Duration of time in milliseconds that the alert will be visible. Set to -1 for indefinite.

id

Id for the alert component. If omitted, a unique one is generated.

message

Alternative way of passing in a message to the alert.

minWidth

Minimum width of the alert.

mousePersist

mousePersist: boolean = false

Whether the alert should stay open if the mouse is hovering over it.

type

type: string

The type of the alert. Can be one of warning, success, information, error or null.

width

width: string

Width of the alert.

dismiss

  • dismiss(reason?: any, manualDismiss?: boolean): void

    Dismisses the alert. If the alert was generated via the AlertService, it is removed from the DOM. Otherwise, it sets the display value to none. Fires the onDismiss event.

    Parameters

    • Optional reason: any

      Data to pass back to the calling component. Only usable if alert is opened using the Service.

    • manualDismiss: boolean = false

      Set to true to skip the dismiss animation.

    Returns void
