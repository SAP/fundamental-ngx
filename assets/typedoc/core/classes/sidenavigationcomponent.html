Selector: <fd-side-nav>

The side-navigation is a wrapping component representing a left navigation that can always display or expand/collapse using the menu icon within the global navigation.

Hierarchy

  • SideNavigationComponent

Implements

  • AfterContentInit
  • AfterViewInit
  • OnInit

Inputs

collapseWidth

collapseWidth: number

The screen width, in pixels, at which to automatically collapse the side navigation on window resize.

condensed

condensed: boolean = false

Whether condensed mode is included

sideNavigationConfiguration

sideNavigationConfiguration: SideNavigationModel

Side navigation configuration, to pass whole model object, instead of creating HTML from scratch

selectable

  • set selectable(selectable: boolean): void

    Whether clicking on elements should change selected state of items

    Parameters

    • selectable: boolean

    Returns void

Properties
