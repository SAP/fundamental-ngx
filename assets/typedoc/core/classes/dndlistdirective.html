dropMode

dropMode: FdDndDropType = 'shift'

Defines drop strategy:

  • shift mode will create line after the closest drop element.
  • group mode will create replace indicator on a whole closest drop element.
  • auto mode will create a line after the closest drop element if dragged element coordinates are shifted for 30% from the center of the closest drop element. Otherwise, it will create replace indicator on a whole closest drop element.

shift mode is the default.

  • Defined in libs/cdk/src/lib/utils/drag-and-drop/dnd-list/dnd-list.directive.ts:55
