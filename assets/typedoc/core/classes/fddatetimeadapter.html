toIso8601

  • toIso8601(fdDate): string

  • Gets the ISO 8601 date-time string for the given date.

    Returns a local (timezone-unaware) string in the format YYYY-MM-DDTHH:mm:ss. This is intentionally not RFC 3339 compliant because no timezone designator is appended.

    This method is used to generate date strings that are compatible with native HTML attributes such as the min or max attribute of an <input>.

    To produce an RFC 3339 string, append a timezone designator to the result:

    const rfc3339Utc = adapter.toIso8601(date) + 'Z';          // UTC
    const rfc3339Local = adapter.toIso8601(date) + '+05:30';    // with offset

    Parameters

    • fdDate: FdDate

      The date to get the ISO date string for.

    Returns string

    The ISO 8601 date-time string without timezone designator.

    Overrides DatetimeAdapter.toIso8601
