Selector: fdScrollSpy

A directive designed to help navigation elements determine the element currently in view of the user.

Hierarchy

  • ScrollSpyDirective

Inputs

fireEmpty

fireEmpty: boolean = false

Whether events are still fired if there is no tag present on the user's screen.

scrollSpyDisabled

scrollSpyDisabled: boolean = false

Whether to disable scroll spy

targetOffset

targetOffset: number = 0

Number that represents the offset in pixels for fired target. 100 value means that the event will be fired for target that is 100 pixels below the spy container.

targetPercent

targetPercent: number = 0

A number that represent at what location in the container the event is fired. 0.5 would fire the events in the middle of the container, 0 for the top and 1 for the bottom.

trackedTags

trackedTags: string[] = []

An array of tags to track.

Outputs

Readonly spyChange

spyChange: EventEmitter<HTMLElement> = ...

Event fired on the scroll element when a new item becomes activated by the scrollspy . The returned value is the HTMLElement itself.
