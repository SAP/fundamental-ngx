Configuration for UI5 Web Components content density markers. UI5 Web Components use data-ui5-compact-size attribute for compact mode. Cozy is the default mode (no marker needed).

Hierarchy

  • Ui5ContentDensityMarkers

Properties

enabled

enabled: boolean

Whether to apply UI5 content density markers. When enabled, data-ui5-compact-size attribute will be applied for COMPACT and CONDENSED modes (UI5 only supports cozy/compact).

Default
true
