columnLayout

columnLayout: Nullable<string>
description

Specify the column layout in the format XLn-Ln-Mn-Sn where n is the number of columns and can be different for each size. eg: XL2-L2-M2-S1 would create 2-column layouts for XL, L, and M sizes and single-column layout for S size.
