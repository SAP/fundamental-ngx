Selector: <fdp-table>

The component that represents a table. A table is a set of tabular data. Line items can support data, images and actions.

```html <fdp-table [dataSource]="source" fdCompact selectionMode="multiple" emptyTableMessage="No data found">

<fdp-table-toolbar title="Order Line Items" [hideItemCount]="false">
