• range<T>(length, mapFn?): T[]

  • Creates an array of numbers from 0 to n-1, optionally transformed by a map function. Useful for iterating a fixed number of times in templates with @for or generating transformed sequences.

    Type Parameters

    • T = number

    Parameters

    • length: number

      The number of elements in the array

    • mapFn: ((index) => T) = ...

      Optional function to transform each index into a value

        • (index): T

        • Parameters

          • index: number

          Returns T

    Returns T[]

    An array of numbers from 0 to n-1, or transformed values if mapFn is provided

    Example
    // Simple iteration (returns [0, 1, 2])
    range(3)

    // With transformation (returns ['Item 0', 'Item 1', 'Item 2'])
    range(3, i => `Item ${i}`)

    // In component for template iteration:
    readonly loadingRange = range(3);

    // In template:
    For

    (i of loadingRange; track i) {

    Item {{ i }}
    }
