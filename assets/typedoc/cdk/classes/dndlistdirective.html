dropMode

dropMode: FdDndDropType = 'shift'
  • Defined in libs/cdk/src/lib/utils/drag-and-drop/dnd-list/dnd-list.directive.ts:42

Defines drop strategy:

  • shift mode will create line after closest drop element.
  • group mode will create replace indicator on whole closest drop element.
  • auto mode will create line after closest drop element, if dragged element coordinates are shifted for 30% from center of the closest drop element. Otherwise, it will create replace indicator on whole closest drop element.

shift mode is the default.
