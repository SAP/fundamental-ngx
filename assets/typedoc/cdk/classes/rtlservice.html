Service for managing RTL (Right-to-Left) state.

Uses signals for reactive state management and is fully compatible with zoneless change detection. The language list is used to determine if RTL should be enabled at startup based on the browser's language. Users can override the default RTL languages by providing the RTL_LANGUAGE injection token.

Example
// Reading RTL state
readonly isRtl = computed(() => this._rtlService.rtl());

// Updating RTL state
this._rtlService.rtl.set(true);
