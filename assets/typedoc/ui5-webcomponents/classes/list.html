accessibilityAttributes

accessibilityAttributes: InputSignal<undefined | ListAccessibilityAttributes> = ...

Defines additional accessibility attributes on different areas of the component.

The accessibilityAttributes object has the following field:

  • growingButton: growingButton.name, growingButton.description.

The accessibility attributes support the following values:

  • name: Defines the accessible ARIA name of the growing button. Accepts any string.

  • description: Defines the accessible ARIA description of the growing button. Accepts any string.

Note:* The accessibilityAttributes property is in an experimental state and is a subject to change.

Readonly slots

slots: {
    description: string;
    name: string;
}[] = ...

Available slots for content projection in this component.

Slots allow you to insert custom content into predefined areas of the web component. Use the slot attribute on child elements to target specific slots.

  • (default): Defines the items of the component.

Note:* Use ui5-li, ui5-li-custom, and ui5-li-group for the intended design.

  • header: Defines the component header.

Note:* When header is set, the headerText property is ignored.

Type declaration

  • description: string
  • name: string
Example
<ui5-list>
  <div slot="header">Custom header content</div>
  <p>Default slot content</p>
</ui5-list>
See

MDN Web Components Slots
