Readonly slots

slots: {
    description: string;
    name: string;
}[] = ...

Available slots for content projection in this component.

Slots allow you to insert custom content into predefined areas of the web component. Use the slot attribute on child elements to target specific slots.

  • (default): Defines the content of the component. The content is visible only when the component is expanded.
  • header: Defines the component header area.

Note:* When a header is provided, the headerText property is ignored.

Type declaration

  • description: string
  • name: string
Example
<ui5-panel>
  <div slot="header">Custom header content</div>
  <p>Default slot content</p>
</ui5-panel>
