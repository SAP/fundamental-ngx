Readonly slots

slots: {
    description: string;
    name: string;
}[] = ...

Available slots for content projection in this component.

Slots allow you to insert custom content into predefined areas of the web component. Use the slot attribute on child elements to target specific slots.

  • (default): Defines the items of the ui5-mcb-item-group.
  • header: Defines the header of the component.

Note:* Using this slot, the default header text of group and the value of headerText property will be overwritten.

Type declaration

  • description: string
  • name: string
Example
<ui5-mcb-item-group>
  <div slot="header">Custom header content</div>
  <p>Default slot content</p>
</ui5-mcb-item-group>

wrappingType

wrappingType: InputSignal<undefined | "None" | "Normal"> = ...

Defines if the text of the component should wrap when it's too long. When set to "Normal", the content (title, description) will be wrapped using the ui5-expandable-text component.

The text can wrap up to 100 characters on small screens (size S) and up to 300 characters on larger screens (size M and above). When text exceeds these limits, it truncates with an ellipsis followed by a text expansion trigger.

Available options are:

  • None (default) - The text will truncate with an ellipsis.
  • Normal - The text will wrap (without truncation).
