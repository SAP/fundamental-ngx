itemsFitContent

itemsFitContent: InputSignalWithTransform<boolean, unknown> = ...

Determines whether the segmented button items should be sized to fit their content.

If set to true, each item will be sized to fit its content, with any extra space distributed after the last item. If set to false (the default), all items will be equally sized to fill the available space.

Readonly slots

slots: {
    description: string;
    name: string;
}[] = ...

Available slots for content projection in this component.

Slots allow you to insert custom content into predefined areas of the web component. Use the slot attribute on child elements to target specific slots.

  • (default): Defines the items of ui5-segmented-button.

Note:* Multiple items are allowed.

Note:* Use the ui5-segmented-button-item for the intended design.

Type declaration

  • description: string
  • name: string
Example
<ui5-segmented-button>
  <div slot="header">Custom header content</div>
  <p>Default slot content</p>
</ui5-segmented-button>
