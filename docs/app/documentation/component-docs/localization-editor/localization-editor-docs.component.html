Basic usage of Localization Editor, to add some text on add-on there is [label] input.

Forms in Localization Editor

Forms usage inside localization editor.

Template in labels

Instead of label there is also way to put whole template, which will be shown in add-on. To achieve it, it's needed to put ng-template with fd-localization-editor-label directive.

Textarea

To use textarea, it's mandatory to add [type]="'textarea'" configuration to wrapper component.
