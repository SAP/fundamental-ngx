Text add-on

The Input with text add-on component is used mainly to specify the type of the data being entered, like: currency, unit of measure, etc. The informational add-on can be left or right aligned.

Icon add-on

An icon can be used instead of text by providing a 'glyph' attribute.

Button add-on

Adding [button]="true" to the component turns either the icon add-on or text add-on into a button, which emits an event addOnButtonClicked($event).

Compact mode

Adding [compact]="true" to the component makes the component compact.

<fd-input-group-number>

The user can increase or decrease a number value using the spinner add-on.

<fd-input-group-search>

This input group automatically shows a clear search button when the user has entered a value in the input.

<fd-input-group-search> Component Within Angular Reactive Form

It is possible to use the input-group-search component within an Angular Reactive Form.
