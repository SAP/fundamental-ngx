Loader element

The loading element is used to display the loading indicator animation.

Usage with other elements

The loading indicator is most often used within another component to indicate the loading state. A container (even ui-, app-, and page-level) can be “activated” by setting aria-busy="true". The spinner should be included inside the container. The container's busy state and the spinner's loading state should always be the same, meaning if the spinner's loading property is true, the parent container's [aria-busy] should also be true.
