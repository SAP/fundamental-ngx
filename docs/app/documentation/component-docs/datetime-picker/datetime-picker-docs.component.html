Simple Datetime Picker

The simplest form of the datetime picker. It makes use of ngModel to bind to the desired value. The alternative is to use the date input combined with the dateChange output. There is also added [position] attribute which allows us to decide where popup should be shown The value for the input only changes the model on blur or an enter/return keystroke.

Non-Meridian Datetime Picker

Removing the am/pm from the time component. The string representation may or may not conserve it depending on locality.

Programmatic Change

It is possible to programmatically change the value of the date of the datetime picker and still conserve full functionality.

Null Validity

Null input values are considered valid by default. Use [allowNull]="false" to make them invalid.

Formatting

Providing a custom format for the dates is possible through providing a service.
