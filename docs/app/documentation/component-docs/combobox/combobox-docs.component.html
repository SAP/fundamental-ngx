Standard Combobox

The standard version of the component that most people will likely use regularly.

Custom Filter

The input supports custom filters through the filterFn input function. Simply pass a function which accepts an array and a search term, and then returns the desired filtered array.

Display Object Property

Passing object arrays to the Combobox is supported through the displayFn input function. To access the this keyword inside the passed function, use an arrow function.

Observable Async Example

The dropdownValues input can accept an Observable with the async pipe, which could be useful for autocomplete. This example also demonstrates how to create a callback function from data returned by the observable.

Custom Item Template

The way the items are rendered in the popover list can be customized through providing an itemTemplate to the component.
