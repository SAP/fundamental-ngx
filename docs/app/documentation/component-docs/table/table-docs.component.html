Table

Column sorting is achieved using the fdColumnSortable directive. Place this directive on the desired column(s) to sort, set the [sortDir] input to the corresponding component variable that dictates sort direction, and use (click) to call the sorting algorithm on your component. Valid values for sort direction are 'asc', 'desc', 'dsc' or 'none'. Note in this example that calling a sort function for one column will reset the sort direction for the other sortable column.
