Calendar
Commonly used as the contents of a popover when composing “date-picker”, rarely used on its own as a standalone component.

The calendar header buttons are navigable with the tab key. Pressing enter or the space bar when the month or year is focused will change the day selection grid to the month or year selection grid respectively. The user can then tab into the day/month/year selection grid and use the arrow keys to navigate around the grid and make a selection with the space bar or enter key.

Calendar with single selection, disabled and blocked days

You can use two-way data binding to select a date or range of dates on a calendar. Calendars internally utilize an interface called CalendarDay, which contains the property date. To select a date you will create a new object that contains the property date, whose value is a native JavaScript Date object. Then, using two-way binding syntax, set the selectedDay value to your new object. Note that selecting a day (or range of days) in this manner will override any block/disable functions.
