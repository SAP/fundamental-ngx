Alert
Alerts are used to display short messages that require the user's attention.

<fd-alert>

Alerts can be opened one of two ways, either by passing a template reference variable or by passing a component reference as content. For simple cases where only one alert will be shown at a time, we recommend the former (see example 1). However, when multiple alerts will be shown simultaneously, we recommend the latter (example 2), as a new instance of the alert component will be created each time the alertService.open function is called. Using the latter method allows reuse of the same alert component with different parameters and will result in better alert stacking behavior.
