Toggle
The Toggle component is used to activate or deactivate an element. It uses a visual metaphor to inform the user of the state of the toggle.

Once focused, it can be toggled using the spacebar or the enter key.

<fd-toggle>

Toggle Sizes

Options include 'xs', 's', 'l' and default. If no value is provided, default is used.

Disabled State

Toggle can be in the disabled state, which prevents the user from interacting with it.
