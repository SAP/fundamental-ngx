Popover Directive

The popover directive is used by the fd-popover component. It constitutes an abstraction of both Popper.js and Fundamental-ngx features.

In cases where the popover component is insufficient, or even unneeded, this directive can serve as a powerful alternative.

fdPopover

Simple Popovers

Creating a popover is simple. Use a string or template as the content of the directive.

Different Triggers

The directive provides a built-in trigger feature, which listens to the user-defined HTML events and toggles the popover. This can also be implemented manually by simply switching an isOpen property.

Programmatic Triggers

A manual implementation of the built-in triggers is possible. Using two-way binding on the isOpen property also allows the use of both. This flexibility also allows you to completely control the behaviour of the popover.

Fill Control Width

A custom Popper modifier allows the directive to match its popover size to that of the control.
