Time Picker
The Time Picker component allows the user to easily set a time using either an HTML5 time input or the Fundamental NGX Time component.

The time picker supports the same keyboard navigability as the time component. The calendar can be opened by focusing the time picker input field and closed by focusing the calendar icon button and pressing enter or space.

<fd-time-picker>

24-Hour Time Picker

This is the default time picker setting.

12-Hour Time Picker with AM/PM

Set [meridian]="true" to use a 12-hour clock..

Time Picker with No Seconds

Use [displaySeconds]=false to turn off seconds for the time picker.

Disabled Time Picker

Set [disabled]="true" to disable the time picker.

Compact Time Picker

Set [compact]="true" to use compact time-picker.
