Simple List

A simple list. The list item can contain plain text or links.

Action List

The list items can also contain quick actions, such as: remove, edit, copy, etc.

Single Select List

Some lists may require the user to only select one item. In those cases, a radio button is the best-fitting input. There is no formal support for this feature, though an example on how to accomplish it is provided below.

Checkbox List

Checkboxes can be used, allowing the user to select multiple items for bulk actions.

Infinite Scroll List

It is possible to implement an infinite scroll function using the fdInfiniteScroll directive. It allows you to load more data as the user scrolls down the parent element. It is important to remember that the InfiniteScrollModule is required for this feature.
