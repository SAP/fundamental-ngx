List
The List Component can be used to display a list of items with simple information such as scopes, names, etc.

List item anchors will be keyboard navigable via the tab key provided they have [routerLink] or [attr.href] attributes. List action buttons and checkboxes can be navigated via the tab key. List checkboxes can be selected with the space bar.

<fd-list>

A simple list. The list item can contain plain text or links.

The List items can also contain quick actions, such as: remove, edit, copy, etc.

Checkboxes can be used, allowing the user to select multiple items for bulk actions.
