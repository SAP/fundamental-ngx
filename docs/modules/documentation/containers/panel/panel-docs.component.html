<fd-panel>

Panel with edge bleed

Use [bleed]="true" on the panel body to have the contents of the panel body bleed to the outer edge. This is useful for components like tables and trees.

Default Panel Grid (3 columns)

Panel Grid with [nogap]

[nogap]="true" will remove the margins between the panels.

Panel Grid with 2 columns

The [col]="2" renders a 2 column grid.
