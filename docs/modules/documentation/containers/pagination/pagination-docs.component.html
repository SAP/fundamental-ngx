Pagination
Use pagination components in conjunction with Lists, Tables, and Trees to allow for more efficient performance and quick access to specific records in the overall data set.

Pagination links are navigable via the tab key and selected with the space bar or enter key.

The pagination component's currentPage value must be updated from your component for the component to work properly.

Pages can be selected programmatically by calling the goToPage(pageNumber) function on the pagination component.
