Breadcrumb
The breadcrumb allows users to see the current page and navigation path to that page. Users can navigate to previous levels in the path by clicking on those levels. By clicking on the current page, a dropdown allows users to access other pages at that same level.

<fd-breadcrumb>


Instead of the [url] property, you may use any RouterLink properties on each breadcrumb item. See the official Angular documentation for RouterLink for more information. An example using routerLink Breadcrumb Level 1 Breadcrumb Level 2 Breadcrumb Level 3
