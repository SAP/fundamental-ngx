Dropdown
The dropdown component lets the user select one of several different options. It is more flexible than the normal Select.

<fd-dropdown>

The dropdown is designed to resemble other Fundamental input components. The options can be divided in groups, which are visually separated and can have a small group header text.

An disabled dropdown is styled differently and is inactive.
