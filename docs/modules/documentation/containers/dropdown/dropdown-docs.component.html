Dropdown
The dropdown component is an opinionated composition of the “fd-popover” and “fd-menu” components with the use of a styled button. It allows users to make one selection from a list. It is more flexible than the normal Select. Generally, it should be used when there are between 3 to 10 or more options.

This component is completely composed from other components and doeszn’t have any of its own.

Default Dropdown

The dropdown is designed to resemble other Fundamental input components. The options can be divided in groups, which are visually separated and can have a small group header text.

Contextual Menu
The contextual menu component is an opinionated composition of the “fd-popover” and “fd-menu” components with the use of a styled button. A More icon or the word “More”, is used to indicate there are more options than room to display them. On click or tap, a contextual menu opens.
