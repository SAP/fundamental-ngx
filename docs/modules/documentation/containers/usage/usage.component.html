Usage

You have the option of importing all of Fundamental NGX at once, or you can import individual components as you need them. To import the whole library, add:

import {{ '{' }} FundamentalNgxModule } from 'fundamental-ngx';

to your app module definition. Also add FundamentalNgxModule to the imports array in the @NgModule declaration.

To include an individual Angular Fundamental component in your application, you only need to import the relevant module. For example, if you wanted to use Alerts, you would add:

import {{ '{' }} AlertModule } from 'fundamental-ngx';

Then add AlertModule to the imports array in the @NgModule declaration.

Lastly, simply add the component to your HTML!
