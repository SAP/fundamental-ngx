Installation

Dependencies

Fundamental NGX has peer dependencies on fiori-fundamentals and ng-bootstrap

NPM

npm install --save fiori-fundamentals @ng-bootstrap/ng-bootstrap

Yarn

yarn add fiori-fundamentals @ng-bootstrap/ng-bootstrap

Download Fundamental NGX

NPM

npm install --save fundamental-ngx

Yarn

yarn add fundamental-ngx

Add Fiori Fundamentals CSS

Fundamental NGX utilizes Fiori Fundamentals styling. You will need to include the Fiori Fundamentals css in your project. Open your angular.json file and add the following to the styles arrays: 
"./node_modules/fiori-fundamentals/dist/fiori-fundamentals.css"

Note the path may be different if your CLI configuration is not in the root of your project directory or if you have set a custom root.

Now you're ready to use Fundamental NGX!
