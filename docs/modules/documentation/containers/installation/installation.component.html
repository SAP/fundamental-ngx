Installation

Dependencies

Fundamental NGX has a few peer dependencies you'll need to add to your project.

NPM

npm install --save @ng-bootstrap/ng-bootstrap fundamental-ui@1.2.2

Yarn

yarn add @ng-bootstrap/ng-bootstrap fundamental-ui@1.2.2

Download Fundamental NGX

NPM

npm install --save fundamental-ngx

Yarn

yarn add fundamental-ngx

Add Fundamental UI CSS

Fundamental NGX utilizes Fundamental UI styling. You will need to include the fundamental ui css in your project. Open your angular.json file and add the following to the styles arrays: 
"./node_modules/fundamental-ui/dist/fundamental-ui.css"

Note the path may be different if your CLI configuration is not in the root of your project directory or if you have set a custom root.

Now you're ready to use Fundamental NGX!
