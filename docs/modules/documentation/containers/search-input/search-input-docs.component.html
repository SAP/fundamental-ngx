Custom Filter

The search input can be used with a custom filter that is decided outside the component.

To accomplish this, you will need to set the [usingCustomFilter] flag to true, like shown below. To display nothing when the user first clicks inside, simply set the starting Subject value to an empty array.

Observable Async Example

The dropdownValues input can accept an Observable with the async pipe, which could be useful for autocomplete. This example also demonstrates how to create a callback function from data returned by the observable.
