Button
The Button directive is used to enhance standard HTML buttons.

<button fd-button>

The inputs for button will be translated into class attributed respecting fundamental-ui guidelines.
For example:
if it is passed positive for fdType the class fd-button--positive will be added to the element.
if it is passed s for size the class fd-button--s will be added to the element.

Extensibility

The button implementation allows certain extensibility in terms of options for the inputs. button inputs doesn't have a restriction for the predefined values.
The predefined values for the input size are xs, s, l, but size can accept any other string, for example xxs, which will be translated into class fd-button--xxs.

Button Types

The default button type is "primary", no modifier is required. Use [fdType]="'main'", [fdType]="'secondary'", [fdType]="'toolbar'", [fdType]="'positive'" or [fdType]="'negative'" to modify the type of the button.

Button Sizes

Use [size]="'xs'", [size]="'s'", [size]="'compact'" or [size]="'l'" to modify the size of the button.

Buttons with Icons

Use [glyph]="'{{ '{' }}icon-name}'" to add an icon to the button.

Button State

Use [state]="'disabled'" or [state]="'selected'" to set button state.
