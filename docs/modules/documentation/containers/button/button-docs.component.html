Button
The Button directive is used to enhance standard HTML buttons.

<button fd-button>

Button Types

The default button type is "primary", no modifier is required. Use [type]="'main'", [type]="'secondary'", [type]="'toolbar'", [type]="'positive'" or [type]="'negative'" to modify the type of the button.

Button Sizes

Use [size]="'xs'", [size]="'s'", [size]="'compact'" or [size]="'l'" to modify the size of the button.

Buttons with Icons

Use [glyph]="'{{ '{' }}icon-name}'" to add an icon to the button.

Button State

Use [state]="'disabled'" or [state]="'selected'" to set button state.
