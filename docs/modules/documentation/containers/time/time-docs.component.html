24-Hour Clock

This is the default clock setting.

12-Hour Clock

You can use a Meridian 12-hour clock by setting [meridian]="true". The passed time object will still contain a 24-hour clock.

Clock Without Spinners

Use [spinners]="false" to hide the spinners.

Clock Without Seconds

Use [displaySeconds]="false" to hide the seconds input.

Disabled State

You can disable the entire component by setting [disabled]="true".
