Image
When using images you can use our helpers classes to adjust the size and the shape. The inputs for fd-image will be translated into class attributed respecting fundamental-ui guidelines.
For example:
if it is passed s for size the class fd-image--s will be added to the element.

Extensibility

The image implementation allows certain extensibility in terms of options for the inputs. fd-image inputs doesn't have a restriction for the predefined values.
The predefined values for the input size are s, m, l, but size can accept any other string, for example xxs, which will be translated into class fd-image--xxs.
