Basic Tab Container The Tab Container provides a tabbed interface for organizing and displaying content in separate sections. Users can click on tabs to switch between different content areas. Tabs can include text, icons, and additional text to provide context. Tab Layout The tabLayout property controls how tab content and additional text are displayed. Use Standard (default, vertical layout) or Inline (horizontal layout) to arrange the tab content and additional text. Tab Design Colors Individual tabs support design colors to provide visual distinction: Default, Neutral, Positive, Critical, and Negative. Overflow Mode When there are many tabs, the overflowMode property controls how overflow is handled. Use End (default) to show a "More" button at the end, or StartAndEnd to show overflow buttons at both ends while preserving tab order. Background Design Customize the appearance of the tab container header and content using headerBackgroundDesign and contentBackgroundDesign properties. Available designs: Solid (default), Transparent, and Translucent. Reorderable Tabs The Tab Container supports drag-and-drop reordering of tabs when the movable property is set to true. Users can click and drag tabs to rearrange their order within the tab container.
