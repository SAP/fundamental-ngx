Basic Sample The Multi ComboBox component allows users to select multiple items from a predefined list with typeahead suggestions. It combines the functionality of a multi-select dropdown with text input filtering. Value State The MultiComboBox can display different value states by setting the valueState property. Disabled and Readonly The MultiComboBox can be set to a disabled or readonly state, preventing user interaction. Grouped Items The MultiComboBox supports grouping of items, allowing for better organization and easier selection. Clear Icon The MultiComboBox can show a clear icon, visible when there is a value, typed by the user. When pressed, the value gets cleared. Select All Items

When showSelectAll is enabled, "Select All" checkbox is displayed to allow users select all item at once.

Text Wrapping

The sample demonstrates how the text of the items wrap when too long. Note: their representation in the input field (known as token) will truncate.

Selection Trigger The ui5SelectionChange event includes a trigger property indicating the source of the selection:
  • typeahead - Selection caused by typeahead (auto-complete while typing)
  • click - Selection caused by clicking or tapping an item in the dropdown
  • keyboard - Selection caused by keyboard navigation (Arrow keys, Home, End, Page Up/Down)
The noTypeahead property disables auto-selection when typing. With this enabled, selection only occurs on explicit user action (Enter key, click, or keyboard navigation). Custom Items with Rich Content Use ui5-mcb-item-custom to create custom item templates with complex layouts, multiple icons, images, or any HTML content. This allows you to build rich, visually appealing dropdown items beyond the standard text format.
