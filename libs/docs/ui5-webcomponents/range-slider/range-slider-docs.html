Basic Range Slider The Range Slider allows users to select a value range by dragging two handles along a track. The startValue and endValue properties define the current range selection, while min and max define the overall boundaries. Use the ui5Change event to respond when the user finishes adjusting the range, or ui5Input for real-time updates during dragging. Step Size and Tickmarks Control the granularity of value selection using the step property. Enable visual tickmarks with showTickmarks. Additionally, you can display labels at intervals using labelInterval. This helps users understand the scale and make precise selections. Tooltips and Editable Values Display tooltips showing current values by setting showTooltip to true. For enhanced accessibility and usability, enable editableTooltip to allow users to directly input values into the tooltip fields. This is particularly useful for precise value entry. Disabled State The Range Slider can be disabled by setting the disabled property to true. In this state, users cannot interact with the slider, and it provides visual feedback that the control is not available.
