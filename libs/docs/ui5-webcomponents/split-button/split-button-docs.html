Basic Split Button The split button provides a primary action along with a dropdown menu of additional options. Click the main button to trigger the primary action, or click the arrow to open a dropdown menu with alternative actions. Design Variants Split buttons support multiple design variants to indicate different semantic meanings:
  • Emphasized (default) - For the primary action on the page
  • Positive - For positive or approval actions
  • Negative - For negative or destructive actions
  • Attention - For actions that require user attention
  • Transparent - For less prominent actions
 Split Buttons with Icons Split buttons can display icons using the icon property. Icons help users quickly identify the button's purpose. You can use icons with text labels or create icon-only buttons. Menu items can also have icons using the icon property on ui5-menu-item.
