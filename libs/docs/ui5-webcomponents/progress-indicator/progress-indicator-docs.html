Basic Sample Basic Progress Indicator with different progress values. The value property accepts a number from 0 to 100 representing the percentage of completion. Custom Display Value and Interactive Controls Progress Indicators can display custom text instead of the default percentage using the displayValue property. You can also hide the value completely with hideValue. This example demonstrates interactive controls to modify the progress indicator properties. Progress Indicator with Value States Progress Indicators can display different semantic states using the valueState property. Available states are: None (default), Positive, Critical, Negative, and Information.
