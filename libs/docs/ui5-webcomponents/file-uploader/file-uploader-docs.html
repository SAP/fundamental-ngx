Basic File Uploader The basic File Uploader allows users to select files from their file system. Click "Browse..." or the button to open the file selection dialog. The component displays the selected file name(s). Multiple Files Enable the multiple property to allow users to select multiple files at once. The component will display all selected file names and provide access to the FileList through the reactive files() computed signal. File Type Restrictions Use the accept property to restrict file types (e.g., ".pdf", ".jpg", ".png"). The property accepts a comma-separated list of file extensions. Remember to include the dot (.) before file extensions. File Size Restrictions Use the maxFileSize property to restrict the maximum file size (e.g., "5MB", "10MB"). Value States The File Uploader supports different value states: None, Negative (Error), Critical (Warning), Positive (Success), and Information. These provide visual feedback about file selection validation or upload status. Custom Button Design Use the hideInput property to hide the default input field and provide a custom button in the default slot. This allows you to create a fully branded file upload experience that matches your application's design system.
