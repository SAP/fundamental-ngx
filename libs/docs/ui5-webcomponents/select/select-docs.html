Basic Select The Select component provides a dropdown menu for single-option selection. Use ui5-option components as children to define the available choices. Control the selected value with the value property, or mark options as selected. The ui5Change event fires when the selection changes. Additional Text The Select component supports displaying additionalText for each option, providing supplementary information alongside the main label. Value States The Select component supports different value states to provide visual feedback: None (default), Success (Positive), Warning (Critical), Error (Negative), and Information. Use value states to indicate validation results or provide contextual information. The required property marks the field as mandatory. Disabled and Read-only States The Select can be disabled or readonly:
  • Disabled: Completely non-interactive and cannot receive focus.
  • Read-only: Not editable but can receive focus and provides visual feedback. In read-only mode, when there is additionalText provided, the select displays the selected value with a customizable separator (controlled by textSeparator).
Custom Options You can define completely custom options via the ui5-option-custom (providing custom content) component. Reactive Forms Integration The Select component integrates seamlessly with Angular Reactive Forms.
