Basic Menu The basic Menu displays a list of items in a popover. Use the opener property to specify the button or element that opens the menu. The menu automatically positions itself relative to the opener element. Menu with Icons Add icons to menu items using the icon property on ui5-menu-item. Icons provide visual cues and improve the user experience by making actions more recognizable. Use SAP icon names from the SAP Icons font library. Hierarchical Sub-menus Create nested menu structures by adding ui5-menu-item elements as children of other menu items. The component automatically displays a sub-menu arrow indicator and handles the sub-menu opening/closing behavior. Sub-menus can be nested to multiple levels. Menu Placement Control where the menu appears relative to its opener using the placement property. Available options are Top, Bottom (default), Left, and Right. The menu automatically adjusts if there's insufficient space in the preferred direction. Additional Text Add additional descriptive text to menu items using the additionalText property on ui5-menu-item. This can be useful for providing more context or information about the action represented by the menu item. Loading State Display a loading indicator inside the menu using the loading property. This is useful when fetching menu items asynchronously. Use loadingDelay to control how long to wait before showing the indicator (default: 1000ms).
