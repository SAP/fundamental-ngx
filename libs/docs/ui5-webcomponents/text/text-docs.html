Basic Sample Max Lines Use the maxLines property to control how many lines the text should display before truncating. Set it to Infinity for unlimited lines, or specify a number to limit the display to that many lines. This is useful for creating consistent layouts and preventing text overflow in constrained spaces. Empty Indicator The emptyIndicatorMode property controls whether an empty indicator is displayed when the text content is empty. Set it to "On" to show a placeholder indicator for empty content, or "Off" to display nothing when content is empty. Custom Styling The text can be customized with pure CSS, applied on the tag.
