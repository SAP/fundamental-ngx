Basic Radio Buttons Radio buttons allow users to select a single option from a group of mutually exclusive choices. All radio buttons with the same name property form a radio button group, where only one can be selected at a time. Use the ui5Change event to respond to user selections. Value States Radio buttons support different value states to provide visual feedback: None (default), Positive (Success), Critical (Warning), Negative (Error), and Information. Use value states to indicate validation results or provide contextual information to users. Disabled and Read-only States Radio buttons can be disabled or readonly:
  • Disabled: Completely non-interactive and cannot receive focus.
  • Read-only: Not editable or selectable but can still receive focus and provides visual feedback.
Text Wrapping Control how the radio button label text behaves when there's not enough space using the wrappingType property.
