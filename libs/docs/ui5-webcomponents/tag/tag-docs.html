Basic Sample The Tag component provides a way to categorize and organize content. It can be used to label items with descriptive keywords and supports various visual states and designs. Tags can be styled with different color schemes and designs to match your application's needs. Tag Designs Tag supports various design types including Neutral (default), Set1, Set2, Information, Positive, Negative, and Critical. Each design conveys different semantic meanings and visual emphasis. Interactive Tags Set the interactive property to true to make tags clickable and focusable. Interactive tags emit ui5Click events when clicked and support keyboard navigation. Text Wrapping Control how tag text behaves when it exceeds the available space using the wrappingType property. Normal allows text to wrap naturally, while None truncates overflowing text. Color Schemes Tags support two sets of 10 predefined color schemes each (1-10) that provide visual variety while maintaining consistency. Use different color schemes to categorize content or create visual hierarchy. Tag Sizes Tags support two sizes: S (Small - default) and L (Large). Choose the appropriate size based on your layout requirements and content hierarchy.
