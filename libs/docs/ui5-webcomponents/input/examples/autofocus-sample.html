Autofocus Attribute

The autofocus attribute automatically focuses a component when the page loads or when it appears in a dialog/popover.

Autofocus in Dialogs

The autofocus attribute is particularly useful in dialogs and popovers, where you want to direct the user's attention to a specific input field immediately when the dialog opens.

Usage Notes

  • The autofocus attribute follows standard HTML behavior
  • Only one element per page/dialog should have autofocus
  • Supported components include: ui5-input, ui5-text-area, ui5-button, ui5-select, ui5-combobox, and more
  • In Angular, use autofocus (boolean attribute) or [autofocus]="true"
  • Consider accessibility: autofocus can be disorienting for screen reader users if overused
