Basic Breadcrumbs Overflow Behavior When the links don't fit the available space, they overflow. Separator Styles The separators property determines the visual style of the separator between breadcrumb items. Available options include Slash (default), BackSlash, DoubleBackSlash, DoubleGreaterThan, DoubleSlash, and GreaterThan. Design Variations The design property controls the visual appearance of the last breadcrumb item. The Standard design displays the last item as the current page (bold and without separator), while NoCurrentPage displays all items as regular breadcrumbs with separators.
