Basic Sample Growing Enable the growing property to make the textarea automatically expand with content. Use rows to set the initial size and growingMaxRows to limit maximum expansion. This provides a better user experience by adapting to content size. Character Limits Control text length using maxlength and showExceededText properties. When showExceededText is false, users cannot exceed the limit. When true, exceeded characters are highlighted and counted, useful for soft limits with warnings. Value States TextArea supports different visual states to communicate status to users: None (default), Positive (success), Critical (warning), Negative (error), and Information. These states provide visual feedback for validation and user guidance. TextArea can also be disabled or readonly. Events TextArea emits various events for different interactions: ui5Change (on focus out), ui5Input (on each keystroke or when something is pasted), ui5Select (on text selection), and ui5Scroll (on content scrolling). Use these events for responsive user interactions.
