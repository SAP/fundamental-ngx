Basic CheckBox The CheckBox component allows users to select one or more options from a set. It supports various states and configurations. States CheckBox supports different states including basic, disabled, read-only, required, and indeterminate states. Value States CheckBox supports different value states to indicate the validation status: None (default), Positive, Critical, Negative, and Information. Text Truncation and Wrapping CheckBox supports different text wrapping options. You can choose between normal wrapping (default) or no wrapping with ellipsis truncation. Reactive Forms Integration CheckBox components can be integrated with Angular Reactive Forms. This example demonstrates a nested form structure with an "Accept all" checkbox that uses indeterminate state to reflect partial selection of sub-checkboxes. The form state automatically syncs between parent and child checkboxes.
