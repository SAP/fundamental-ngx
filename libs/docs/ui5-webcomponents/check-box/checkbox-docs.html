Basic CheckBox The CheckBox component allows users to select one or more options from a set. It supports various states and configurations. States CheckBox supports different states including basic, disabled, read-only, required, and indeterminate states. Value States CheckBox supports different value states to indicate the validation status: None (default), Positive, Critical, Negative, and Information. Text Truncation and Wrapping CheckBox supports different text wrapping options. You can choose between normal wrapping (default) or no wrapping with ellipsis truncation. Group of Checkboxes in a Form Checkboxes can be logically grouped using the same name attribute to create a multi-selection form field. When the form is submitted, each checked item will send its individual value to the server, allowing backend processing to identify exactly which options the user selected.
