Basic Icons UI5 Icons provide a comprehensive set of SVG icons from SAP's design system. Icons support different collections including standard SAP icons, TNT icons for technical applications, and Business Suite icons for enterprise scenarios. Icon Designs Icons support different semantic designs to convey meaning through color. The design property controls the visual appearance and semantic meaning of icons. Interactive Icons Icons can be interactive or decorative. Interactive icons can be clicked and focused, while decorative icons are purely visual. The mode property controls this behavior, and interactive icons support click events using Angular 20 signals. Icon Customization Icons can be customized with different sizes, colors, and styles using CSS. SAP Fiori Tools Icons The SAP TNT Icons are available in the @ui5/webcomponents-icons-tnt package. SAP Business Suite Icons The SAP Business Suite Icons are available in the @ui5/webcomponents-icons-business-suite package.
