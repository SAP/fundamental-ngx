Basic Sample The Message Strip component provides a way to display contextual information to users. It can be used to communicate status, warnings, errors, or general information in a non-intrusive manner. Message Strip Designs Message Strip supports various design types including Information, Positive, Critical, and Negative. Additionally, it supports ColorSet1 and ColorSet2 designs with different color schemes. Hide Icon and Close Button

Use hideIcon and hideCloseButton properties to hide some parts of the MessageStrip.
