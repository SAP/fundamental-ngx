Slider The Slider component allows users to select a value from a range of values by moving the slider thumb. It can be used to adjust settings such as volume, brightness, or any other parameter that requires a range of values. Custom Tickmarks The tickmarks property allows you to define custom labeled markers on the slider scale. Each tickmark has a numeric value and an optional label. When the slider value matches a tickmark, the label is shown in the tooltip and announced via aria-valuetext.
