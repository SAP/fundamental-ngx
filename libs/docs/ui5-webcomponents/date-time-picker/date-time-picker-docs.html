Basic Date Time Picker The basic Date Time Picker allows users to select both date and time values. Click the input field to open the popover with calendar and time selectors. The component automatically formats the selected date and time according to the locale. Value States The Date Time Picker supports different value states: None, Negative, Critical, Positive, and Information. These provide visual feedback to users about the validity or status of their input. Min and Max Dates Use the minDate and maxDate properties to restrict the selectable date range. Dates outside this range will be disabled in the calendar. Format Patterns The Date Time Picker allows you to customize the date and time format using the formatPattern, displayFormat, and valueFormat properties. This enables localization and different display preferences. Supported format options are pattern-based on Unicode LDML Date Format notation. For more information, see UTS #35: Unicode Locale Data Markup Language. Events The Date Time Picker emits various events: ui5Change fires when the user confirms a value change, ui5Input fires on every keystroke, ui5Open and ui5Close track the popover state, and ui5ValueStateChange can be used to intercept value state changes. Accessibility The Date Time Picker provides comprehensive accessibility support through ARIA attributes. Use accessibleName to provide a label, accessibleDescription for additional context, and their *Ref counterparts to reference existing DOM elements.
