Basic Example The UI5 Title component is used to display headings and titles in different semantic levels and visual sizes. It provides a simple way to create consistent typography across your application while maintaining proper semantic hierarchy for accessibility. Visual Sizes The size property controls the visual appearance and font size of the title. Available sizes range from H1 (largest) to H6 (smallest). The size property is independent of the semantic level, allowing for flexible design while maintaining proper accessibility. Semantic Levels The level property defines the semantic importance of the title for screen readers and SEO. Available levels are H1 through H6, with H1 being the most important. This property affects the underlying HTML element but not the visual appearance. Text Wrapping The wrappingType property controls how text behaves when it doesn't fit in the available space. Use "Normal" for standard text wrapping or "None" to prevent wrapping and allow text to overflow.
