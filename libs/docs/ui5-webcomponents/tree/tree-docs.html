Basic Tree The basic tree displays hierarchical data with expand/collapse functionality. Use ui5-tree-item components as children to create the tree structure. Nest tree items to create parent-child relationships. Selection Modes The selectionMode property controls how items can be selected. Available modes include None, Single, SingleStart, SingleEnd, Multiple, and Delete. The ui5SelectionChange event fires when selection changes. Tree with Icons and Additional Text Tree items can display icons using the icon property. Icons help users quickly identify different types of nodes in the tree structure, such as folders, files, or different data types. Also, you could add some clarification text via the additionalText property. Lazy Loading For large tree structures, use the hasChildren property to indicate nodes that have children without loading them initially. Load child nodes dynamically when the user expands a node by listening to the ui5ItemToggle event. Call preventDefault() to control the expansion behavior. Custom Content Use ui5-tree-item-custom to create tree items with custom interactive content. Instead of simple text, you can place any UI5 component in the content slot, such as buttons, selects, inputs, or other interactive elements. This is useful for creating rich, interactive tree structures with embedded controls. Drag and Drop The tree component supports drag-and-drop functionality for reordering items. Users can drag items to new locations within the tree, and the component will handle the updates automatically.
