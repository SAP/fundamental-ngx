Basic Sample The Date Range Picker component allows users to select a date range from a calendar popup. It displays two dates (start and end) separated by a delimiter. The component provides keyboard navigation, accessibility support, and internationalization capabilities. Show Two Months Use the showTwoMonths property to display two consecutive months side by side in the picker popup. This makes it easier to select date ranges that span multiple months without needing to navigate between them.

Note: On mobile devices, only a single month will be displayed regardless of this setting.

 Custom Delimiter Use the delimiter property to customize the separator between start and end dates. By default, it uses the locale-specific delimiter (typically a hyphen "-"). You can set custom delimiters like " to ", " - ", or any other string. Date Format Patterns Date Range Picker supports various format patterns for displaying dates. Use formatPattern to control how dates appear in the input field. Common patterns include yyyy-MM-dd, dd/MM/yyyy, MM/dd/yyyy, and localized patterns. Min and Max Dates

Use minDate and maxDate properties to restrict the selectable date range. Dates outside this range will be disabled in the calendar popup.

Note: When using minDate and maxDate, ensure the date format matches your formatPattern or use ISO format (yyyy-MM-dd).

Value States Date Range Picker supports different value states to indicate validation status: None, Success, Warning, Error, and Information. Each state displays a different color and icon to communicate the field's status to users. Component States Date Range Picker can be disabled, readonly, or required. Use these properties to control user interaction and indicate mandatory fields.
