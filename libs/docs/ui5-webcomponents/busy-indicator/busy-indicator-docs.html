Basic Busy Indicator The Busy Indicator is used to show that some operation is going on and the user needs to wait. It can overlay content to indicate that the underlying content is busy. Sizes The Busy Indicator supports different sizes: Small (S), Medium (M - default), and Large (L). Text Placement The text can be placed either at the top or bottom of the busy indicator. The default placement is bottom. Delay You can set a delay in milliseconds after which the busy indicator will be shown. This is useful to avoid showing the indicator for very short operations. The default delay is 1000ms.
