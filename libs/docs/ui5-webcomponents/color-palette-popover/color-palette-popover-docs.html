Basic Color Palette Popover Default Color Use the defaultColor property to set a predefined color that will be initially selected in the palette. The default color should be a valid CSS color and match one of the color palette items. Show More & Recent Colors Set showMoreColors to true to display a "More Colors" button that opens a color picker dialog, allowing users to select custom colors beyond the predefined palette. You can also enable showRecentColors to display recently selected colors for quick access.
