Basic Tokenizer Multi-line Tokenizer With multiLine enabled, tokens are displayed across multiple lines for improved readability. Read-only Tokenizer A read-only tokenizer prevents token deletion and provides visual feedback that the component is not editable. Use the readonly property to make the tokenizer read-only. Clear All Button When multiLine is enabled, you can also show a "Clear All" button by setting the showClearAll property. This button allows users to delete all tokens at once.
