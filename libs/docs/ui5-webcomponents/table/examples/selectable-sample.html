Single Selection Table

Implement row selection in the table using the ui5-table-selection-single component. This feature allows users to select one row, and you can handle selection changes via the ui5Change event.

The selection features of the table is key-based, therefore each row requires a row-key attribute to be uniquely identified.

Multi Selection Table

Enable multi-selection in your table by adding the ui5-table-selection-multi component. This feature allows users to select one or more rows, and you can handle selection changes via the ui5Change event.

The selection features of the table is key-based, therefore each row requires a row-key attribute to be uniquely identified.
