Basic Page Background Design The backgroundDesign property controls the background color of the page content area. Three options are available:
  • Solid (default) - Standard background with solid color for general content
  • Transparent - Transparent background that inherits from the parent container
  • List - Optimized background for list content with better color contrast. Use this when placing a ui5-list component inside the page
Scrolling Behavior The noScrolling property controls whether the page content is scrollable. By default, the page content can scroll vertically when it exceeds the available space. When noScrolling is set to true, vertical scrolling is disabled and all content must fit within the visible area. Footer Configuration The Page component provides two properties to control the footer:
  • hideFooter - Controls footer visibility. When set to true, the footer is completely hidden from view
  • fixedFooter - Controls footer positioning. When set to true, the footer is fixed at the very bottom of the page. When false, it floats over the content with a slight offset from the bottom, creating a floating effect
Use these properties to customize the footer behavior based on your application's requirements.
