Basic User Settings Dialog The User Settings Dialog is an SAP Fiori-specific web component designed to be used within the ui5-user-menu, which itself is placed in a ui5-shellbar. It provides a comprehensive interface for users to view and manage their account settings, preferences, and application-specific configurations. The dialog displays a navigation list of settings categories on the left and the corresponding content views on the right.

The component uses ui5-user-settings-item elements in the default slot to define the settings categories. Each item can contain one or more ui5-user-settings-view elements that display the actual settings content. Use the fixedItems slot for settings items that should remain at the bottom of the navigation list (such as "Sign Out" or "Legal Information").

Key features:
  • open: Controls the visibility state of the dialog
  • headerText: Sets the title text displayed in the dialog header
  • showSearchField: Enables a search field to filter through settings items (disabled by default)
  • ui5SelectionChange: Event fired when a different settings item is selected in the navigation
  • ui5Open / ui5Close: Events fired when the dialog opens or closes
This example demonstrates the complete integration flow: ShellBar → User Menu → User Settings Dialog with multiple settings categories including appearance, language, and notification preferences.
