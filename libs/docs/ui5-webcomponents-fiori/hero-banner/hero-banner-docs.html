Basic The Hero Banner component displays a personalized greeting area at the top of a page. It consists of a header area with an overline text and main header text. Columns Ratio The columnsRatio property controls how the content area is divided when using both default and endContent slots:
  • No columnsRatio - Single column layout, content spans the full width
  • Equal - Two equal columns where both content blocks share the available width equally
  • FirstWider - Two unequal columns where the start content takes two-thirds of the width and the end content takes one-third
On smaller screens, both slots stack vertically regardless of the ratio setting. Header Block and Actions Placement The actionsPlacement property controls where the actions slot content is positioned:
  • TopEnd (default) - Actions are displayed to the right of the header text, at the top of the header row
  • BottomStart - Actions are displayed below the header text, left-aligned
The headerBlockPlacement property controls the vertical placement of the header block:
  • Top (default) - Header block is placed at the top of the content area
  • Bottom - Header block is pushed to the bottom of the first column
