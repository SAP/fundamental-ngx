Basic ShellBar The ShellBar provides a consistent header for your application with primary branding and navigation. Use the primaryTitle and secondaryTitle properties to display application titles. The logo slot accepts an avatar or image component for branding. The profile slot can be used to display user profile information. All Features A comprehensive ShellBar demonstrating all available features including notifications, search, content items, and profile. With Search Field The ShellBar can include a search field by placing a ui5-input component in the searchField slot. Set showSearchField to true to display the search field. The search field is responsive and will collapse on smaller screens. Use the ui5SearchButtonClick event to handle search activation and the ui5SearchFieldToggle event to track when the search field is expanded or collapsed. With Embedded Back Navigation The ShellBar can include an embedded back navigation button by placing a ui5-button component in the startButton slot. Trial Example ShellBar configured for trial environments with trial tags and remaining days indicator. ShellBar Events The ShellBar fires various events for user interactions including logo clicks, profile clicks, notification clicks, product switch clicks, menu item clicks, and search-related events. Use these events to implement navigation, display popovers, or trigger other application logic. This example demonstrates tracking all major ShellBar events with a history log.
