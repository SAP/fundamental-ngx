Basic User Menu The User Menu component is an SAP Fiori specific web component designed to be used within a ui5-shellbar. It provides a comprehensive user account management interface, displaying the current user's information with an avatar and allowing users to access various account-related actions. The menu appears when the user clicks on the ShellBar's profile avatar. The opener property should reference the ShellBar's profile element, and the menu is typically triggered via the ui5ProfileClick event. Multiple Accounts The User Menu supports multiple user accounts within the ShellBar context, allowing users to switch between different profiles. Add account items to the accounts slot to enable account switching functionality. The first account or one marked with selected will be shown as the current account.
  • showOtherAccounts: Controls whether the "Other Accounts" section is visible in the menu
  • showEditAccounts: Shows an "Edit Accounts" button for account management
  • ui5ChangeAccount: Event fired when the user switches to a different account
 Menu Items Add custom menu items to the default slot to provide additional user-related actions within the ShellBar's user menu. Use ui5-user-menu-item components to create the menu structure. The ui5ItemClick event is fired when any menu item is selected.
