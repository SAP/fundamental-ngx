Tag Slot Guidelines

  • Tags are visible when the navigation is expanded, and hidden when collapsed
  • In collapsed mode, tags appear in the overflow menu
  • Use design="Set2" and hideStateIcon attributes
  • Use colorScheme values 5-10 to avoid confusion with semantic colors
  • Limit tag width to ~64px (4rem). Use abbreviations for longer text
  • Tags must NOT be interactive (active must be false)
