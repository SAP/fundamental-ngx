Basic Example The View Settings Dialog allows users to sort, filter, and group data. It provides three tabs: Sort, Filter, and Group. Click the button below to open the dialog and configure the settings. Custom Tabs The View Settings Dialog supports custom tabs alongside the built-in Sort, Filter, and Group tabs. Custom tabs are rendered after the built-in tabs and allow application-specific settings (e.g. column visibility, row density). Use the ui5-view-settings-dialog-custom-tab component in the customTabs slot to define additional tabs with custom content. The resetEnabled property and ui5Reset event must be managed by the application since the dialog cannot detect changes in custom tab content.
