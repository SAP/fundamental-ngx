Line Clamp Standard Usage

Use fdkLineClamp directive on a container element with [fdLineClampLines] to set the maximum number of lines and [fdLineclampState] to enable/disable clamping.

Apply fdLineClampTarget to the text element inside the container and pass the text to [fdLineClampTargetText].

The (lineCountUpdate) event emits the actual number of lines the text would occupy without clamping.
