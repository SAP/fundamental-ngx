Local Containers (Independent BidiService Instances)

Each container below has its own BidiService provider. They operate independently.

Note: This example demonstrates two patterns:

  • Global BidiService: Provided at component level, shared by all elements in the "Global Container". The global switch and button control this instance.
  • Local BidiService: Each "Local Container" has its own provider, creating independent instances. Changes in one local container don't affect others or the global container.

The BidiService automatically detects RTL languages (Arabic, Hebrew, Persian, etc.) based on navigator.language. You can manually override the direction at any level.
