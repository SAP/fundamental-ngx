BidiService Basic Example The BidiService provides signal-based reactive state management for bidirectional text direction (RTL/LTR). It integrates with Angular CDK's Directionality service and automatically detects RTL languages based on browser settings. Features
  • Signal-based reactivity: Uses Angular signals for reactive state management
  • Automatic RTL detection: Detects RTL languages based on navigator.language
  • CDK integration: Works seamlessly with Angular CDK Directionality
  • Customizable: Override RTL language list via RTL_LANGUAGE injection token
  • Type-safe: Full TypeScript support with proper typing
