Focusable Item

The [fdkFocusableItem] directive is used to mark elements as focusable items within a focusable list or grid. It manages focus states, keyboard navigation (Enter, F2, Escape), and integrates with screen readers for accessibility announcements. Use it in combination with [fdkFocusableList] or [fdkFocusableGrid] directives to build accessible, keyboard-navigable components.
