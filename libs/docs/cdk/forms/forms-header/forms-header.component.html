Control value accessor

Forms module exposes standalone CvaDirective which implements ControlValueAccessor interface for more productive form-related components development.

Additionally, Forms package exposes models and tokens for form items such as FormField, FormState and FormFieldControl, which are widely used in platform package.

By providing these tokens and implementing respective interfaces, developers can create custom form controls which would be recognizable by Platform's Form Container.

Control value accessor directive supports following features:

  • Both template-driven and reactive forms;
  • Automatic state detection of the control;
  • Value read/write with respective events emitted;
  • Disabled state;
