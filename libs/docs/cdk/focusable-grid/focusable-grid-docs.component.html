Shortcuts
Key Action
Arrow Up Move focus to the previous row
Arrow Down Move focus to the next row
Arrow Left Move focus to the previous cell
Arrow Right Move focus to the next cell
Home Move focus to first cell in a row
End Move focus to last cell in a row
Page Up Move focus to first visible cell in a column and make it last visible one
Page Down Move focus to last visible cell in a column and make it first visible one
F2 Move focus to the cell when row in focus and vice versa
Tab Move focus to the next focusable element when in cell
Shift + Tab Move focus to the previous focusable element when in cell
Basic example

To automatically focus cell's focusable child and remove it from tab sequence, use focusChild input property of the [fdkFocusableItem] directive.

To enable wrapping to move focus to the prev/next row when there are no more items in the row in the given direction, use wrapHorizontally input property of the [fdkFocusableGrid] directive.

It's even possible to have whole focusable row, to achieve this, use focusable input property of the [fdkFocusableList] directive. When focused on the row to dive in and out of cell use F2 key.

Short rows To specify which cell to focus in "short row" (row with less cells than the rest of the rows), use shortRowFocus input property of the [fdkFocusableGrid] directive. It accepts 'first' | 'last' | null, where null means to do not move focus (default).
