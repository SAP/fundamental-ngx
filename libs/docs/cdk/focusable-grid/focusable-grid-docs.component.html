Basic example To automatically focus cell's focusable child and remove it from tab sequence, use focusChild input property of the [fdkFocusableItem] directive.

To enable wrapping to move focus to the prev/next row when there are no more items in the row in the given direction, use wrapHorizontally input property of the [fdkFocusableGrid] directive. Short rows To specify which cell to focus in "short row" (row with less cells than the rest of the rows), use shortRowFocus input property of the [fdkFocusableGrid] directive. It accepts 'first' | 'last' | null, where null means to do not move focus (default).
