Basic initial focus

Initial focus directive is designed to ease setting initial focus on some element.

 Initial focus with nested elements

By default, initial focus directive will try to focus element to which this directive is applied.

If this element is not focusable or tabbable, it will try to find such element in it's children.

Developers can change order of search for the tabbable elements by providing [focusLastElement]="true" input property.

Additionally, developers can specify CSS selector of the focusable element manually, to focus on a specific element.
