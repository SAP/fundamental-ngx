Usage with provider example

In some cases you might not want to include (fdkClicked) event in your application, but want to have that unified event on disposal. For that reason you can include into your component's providers FdkClickedProvider and use it as an output, or as a notifier for your internal needs. This provider will work in both cases, whether (fdkClicked) is available or not.
