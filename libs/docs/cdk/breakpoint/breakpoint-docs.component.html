Basic example

*fdkBreakpoint="[BREAKPOINT_NAME]" BREAKPOINT_NAME can be one of the following: S, M, L, XL, also, if you want to use custom breakpoint you can use lt and gt inputs

Alias naming

You can use alias naming for breakpoints, for example, you can use [fdkBreakpointS] on ng-template instead of *fdkBreakpoint="['S']"
