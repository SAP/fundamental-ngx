Basic example

*fdkBreakpoint="[BREAKPOINT_NAME]" BREAKPOINT_NAME can be one of the following:

BREAKPOINT_NAME Container size
S [0 - {{ breakpointSizes.S }})px
M ({{ breakpointSizes.S }} - {{ breakpointSizes.M }})px
L ({{ breakpointSizes.M }} - {{ breakpointSizes.L }})px
XL ({{ breakpointSizes.L }} - ∞]px

also, if you want to use custom breakpoint you can use lt and gt inputs

 Alias naming

You can use alias naming for breakpoints, for example, you can use [fdkBreakpointS] on ng-template instead of *fdkBreakpoint="['S']"

Alias naming

By default, FdkBreakpoint uses window object to get the width of the screen. If you want to use another source, you can use fdkBreakpointObserve input.
