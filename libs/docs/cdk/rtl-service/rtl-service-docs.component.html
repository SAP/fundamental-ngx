RTL Service Usage The RtlService service can be used to manage the Right-To-Left (RTL) settings in your application. It uses a BehaviorSubject to track the RTL status and provides a Signal wrapper for reactive programming. The service can be configured with a custom list of RTL languages using the RTL_LANGUAGE injection token. {{ injectionTokenExample }} Default RTL Languages The service comes with a predefined set of RTL languages. These are:
  • Arabic (ar)
  • Aramaic (arc)
  • Divehi (dv)
  • Persian (fa)
  • Hausa (ha)
  • Hebrew (he)
  • Khowar (khw)
  • Kashmiri (ks)
  • Kurdish (ku)
  • Pashto (ps)
  • Urdu (ur)
  • Yiddish (yi)
You can override this list by providing a custom list to the RTL_LANGUAGE injection token.
