Wide with Icon, 1 Level Wide Text Only, 1 Level Important: This type of Side Navigation does not have Narrow variant. Wide Text Only, 3 Level, Collapse-Only Important: This type of Side Navigation does not have Narrow variant. In the "Collapse-Only" variants, clicking an item that has a child list only expands/collapses the child items. The parent item is not itself navigable. Wide with Icon, 3 Level, Collapse-Only In the "Collapse-Only" variants, clicking an item that has a child list only expands/collapses the child items. The parent item is not itself navigable. Wide with Icon, 3 Level, Navigation and Collapse In this example, the items that have children are navigable themselves, and the button to the side of the parent items acts separately, expanding or collapsing the child elements. Dynamic Width Side Navigation can react dynamically to long label texts in navigational items. In such case the width of the Side Navigation is adapted by the longest label. The label does not wrap or truncate. Use the input [dynamicWidth] for Side Navigation with dynamic width. Navigation with Filter Input A search input can be added to allow the user to filter through the list of navigation items. The side nav occupies the height of its container, so it is advised to be placed in a container of a fixed height, as in the example below. Narrow For the narrow variant of the side navigation, use the [narrow] input. Navigation with Overflow in Narrow Mode If the side navigation is in narrow mode and it contains more items than can be displayed due to the height of its parent container, overflow buttons will be made visible, allowing the user to vertically scroll through the items. This behavior can be disabled by setting the side navigation's [showScrollButtons] input to false. Navigation with Shellbar To use the CX Side Navigation with fd-shellbar, you must use the [cxSideNav] input on the shellbar. Navigation in Mobile Mode Use the [mobile] input to use the side navigation in mobile mode. This will make the side nav occupy the entire screen. In this example, the side navigation is wrapped in a 320x480px container.
