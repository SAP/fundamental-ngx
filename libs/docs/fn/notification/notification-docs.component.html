Notification Standard Usage

Notifications can be created by using NotificationService.

This service supports creating notifications from the following entities:

  • Use built-in component for rendering plain notification;
  • Use provided TemplateRef to render the notification;
  • Use provided Component Class to render the notification.
 Notification Semantic State

Notifications also support semantic states.

To add semantic state to your notifications, use semantic property of the notification configuration.

Notification Position

By default, notifications are stacked at the top right corner and newly added items will be stacked below the old ones.

To change the position and direction of the notifications, use setNewPositionStrategy method of the NotificationService.

Additionaly, developers can create their own position strategies by extending BaseToastPosition class.
