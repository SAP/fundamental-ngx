Avatar

The Avatar is used for displaying images. These can be user profiles, user initials, icons, product images, etc. Use the Avatar only in case you need to display an image, initials, icons. Do not use the Avatar to include an image for any other case.

Size variations:

  • Extra Small (1.5rem)
  • Small (2rem)
  • Medium (2.5rem)
  • Large (3rem)
  • Extra Large (4rem)

Color variations:

  • Indigo
  • Crimson
  • Cyan
  • Lime
  • Pink
  • Yellow
  • Teal
