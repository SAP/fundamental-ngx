Rounded Decorative Avatar with Initials

Rounded Decorative Avatar is the default type for avatar.

In order to change background color of the avatar, developers can pass [color] input property.

 Circle Decorative Avatar with Initials

Developers can change shape of the avatar to a circle by providing [circle]="true" input property.

Avatar with icon

Avatar component supports displaying icon instead of abbreviation.

To display icon, developers need to pass [glyph] input property.

 Avatar with Background Image

To provide an image resource (url or data:base64) use the [image] property.

 Interactive Avatar with icon

Interactive Avatars have hover, active, focus and disabled states.

To make avatar component interactive, developers need to pass [interactive]="true" input property.

To make avatar disabled, developers need to pass [disabled]="true" input property.

 Default image with alter icon options

To provide alternative icon options by priority use the [alterIcon] input property.

There's few options can be set the order priority, Avatar component will process them and display first successful option if [image] url fails to load:

  • content - Avatar will display first letter from string, provided as component content
  • alt - Avatar will display first letter from string, provided as alt attribute
  • backup - Avatar will display image, provided as backupImage
  • default-icon - Avatar will display user glyph icon

Alternative options can be passed the following way alterIcon="content|alt|backup|default-icon"
