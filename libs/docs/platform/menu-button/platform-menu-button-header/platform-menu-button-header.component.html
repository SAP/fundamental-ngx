Menu Button
This component has been deprecated and will be removed in the next major release. Use @fundamental-ngx/core/button in conjunction with @fundamental-ngx/core/menu instead. The menu button is used for triggering the display of a menu.
In @fundamental-ngx/core, it is an extension of @fundamental-ngx/button directive, but in Platform this is a separate component and it reduces the complexity to create Menu Button.


Platform specific enhancements:

  • Menu Button wrapped up as a component.
  • This component is used in conjunction with the Platform Menu Trigger.
  • Menu management is made easy by abstracting the menu definition and linking it with Menu Trigger.
