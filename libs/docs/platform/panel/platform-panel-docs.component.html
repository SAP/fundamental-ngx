Expandable Panel Expandable panels are much like fixed panels, except their content can be expanded and collapsed. Fixed Panel Fixed panels are useful for grouping custom content. To create a fixed panel, add [expandable]="false" to the Panel. Compact Panel For a compact panel, apply the [fdCompact] directive. Fixed Height Content Panel It's possible to set height of panel content by [contentHeight] option. Panel Actions Buttons and other action elements with in this tag will be placed in the Panel header toolbar Panel Config Use Panel Config to override default options
