Textarea This example shows basic Textarea usage with and without Platform Forms Textarea with growing option These examples show how Textarea behaves when growing option is used. The following properties can be used:

  • growingMaxLines - the number of visible lines. Height of textarea is according to this number.
  • height - if specified, overrides growingMaxLines to set the height of textarea.
  • cols - approximately, the number of characters in a single line when the width of textarea is not set.
  • growing - when set to true, resizes the textarea to either growingMaxLines or height, whichever is applied. As user types, the textarea overflows beyond the width to the next line and grows in height until the number of growingMaxLines is reached, or until the height is reached; whichever takes precedence. Beyond this, a scrollbar appears.
Textarea with message counter This example shows how Textarea behaves when showExceededText is set. It also shows form submission.
  • When showExceededText is disabled, the user is not allowed to exceed the number of characters set in the maxLength property. If the textarea is prefilled with some value that exceeds the maxLength, pressing backpress will erase the exceeded characters to bring the count under limit and user won't be able to type beyond this limit.
  • When showExceededText is enabled, characters exceeding the maxLength value are selected on paste, and the counter below the input field displays the number of characters that are over the limit.
