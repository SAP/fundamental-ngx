Basic Datetime Picker Basic usage of Datetime Picker without Forms and with options like meridian, allowNull, disabled, pre-populating and changing dates. Datetime Picker with Reactive Forms Usage of Datetime Picker with Platform Forms using Reactive Forms. When used with Platform Forms, the Datetime Picker will take 100% of its parent container width depending on the chosen column layout. Datetime Picker with Template Forms Usage of Datetime Picker with Platform Forms using Template Forms. Datetime Picker with [disableFunction] Date Time picker that updates it's value on blur. Set [processInputOnBlur]="true" in order to move the processing of input value to whenever component loses focus. You may also hit "enter" to apply the value Mobile mode To use Datetime picker in mobile mode:
  • Set [mobile] property to true
  • Set [mobileConfig] with MobileModeConfig configuration object to customize the look of the control
  • Set [mobileLandscape]="true" to render Calendar in Landscape mode, or [mobilePortrait]="true" for a Portrait mode.
Datetime picker Mobile Mode can be configured globally by providing MOBILE_MODE_CONFIG token. Check Global Configuration page for more information.
