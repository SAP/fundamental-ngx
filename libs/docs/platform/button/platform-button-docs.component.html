Button Types The default button does not require modifier. Use buttonType="emphasized", buttonType="host", buttonType="positive",buttonType="attention", buttonType="transparent" or buttonType="negative" to modify the type of the button. Use label="text" to give label to the button. Button with Icons Use glyph="icon-name" to add an icon to the button and use label="text" to give label to the button.

glyphPosition="before" before and after are used to place the icons beside label. Button with long text truncated When user specifices width of the button then text will start truncating to accomadate it self in the specified width as shown below.
