Button
This component has been deprecated and will be removed in the next major release. Use @fundamental-ngx/core/button instead. The Button Component in Platform is built on top of button directive present in fundamental-ngx/core.

Platform specific enhancements:

  • Split menu button, menu button and segmented button are provided as separate components.
  • Added feature to set button width along with text truncation.
  • Enhanced accessibility support with screen readers
