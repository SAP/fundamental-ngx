Message Popover

Message Popover is a combination of components that allow to render form errors in a human-readable list.

This component can be used along with such technologies:

  • Template-driven forms
  • Reactive forms forms
  • Form Container component
  • Form Generator component

In order for Message popover to listen for the form errors, developers need to wrap their forms into fdp-message-popover-form-wrapper.

Then place fdp-message-popover anywhere on the page, and pass instance of fdp-message-popover-form-wrapper to it.
