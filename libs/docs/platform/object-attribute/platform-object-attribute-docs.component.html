Object attribute standlone text Use label="text",to set the object attribute as standlone text. Object Attributes with long text truncation By default object attribute text will be truncated when the content is more than the view port avalible. On mouseover user can view the full text in tooltip Object Attributes with link option Use [islink]="true" along with linkedText="linktext". Will help to give option to bind required operation on click of link item.
[disabled]="true" make the link non operable.
