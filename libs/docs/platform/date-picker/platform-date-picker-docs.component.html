Date picker Mobile Mode can be configured globally by providing MOBILE_MODE_CONFIG token. Check Global Configuration page for more information. Calendar Legend The Date Picker can display a legend for special days by configuring the following properties:
  • [specialDaysRules] - An array of SpecialDayRule objects that define which days should be marked with special styling. Each rule contains:
    • specialDayNumber - A number between 1-20 that determines the marker style
    • rule - A function that receives a date and returns true if it should be marked
    • legendText - The descriptive text shown in the legend for this special day type
  • [showCalendarLegend] - Set to true to display the legend below the calendar
  • [legendCol] - Set to true to display the legend in a column layout (vertically), or false for a row layout (horizontally)

How to Use

  1. Define your special day rules in the component using the SpecialDayRule interface
  2. Pass the rules array to the date picker's specialDaysRules input
  3. Enable the legend by setting showCalendarLegend to true
  4. Optionally, set legendCol to true for a vertical column layout

Special Day Properties

The specialDayNumber property can be any number from 1 to 20, with each number corresponding to a different visual marker style. Common uses include marking holidays, deadlines, meetings, or other important dates.
