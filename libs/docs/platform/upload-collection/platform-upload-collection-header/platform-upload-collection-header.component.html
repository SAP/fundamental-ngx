Upload Collection

The upload collection allows users to upload one or more files from different devices.

Usage

Use the upload collection if:

  • You want to show a list of uploaded files that can be modified.
  • You want to allow users to add or remove files, and to change the file names.

Don't use the upload collection if:

  • The user can upload only one file to the app. In this case, use the FileUploader control instead.

Required

  • To use all CRUD operations need to describe UploadCollectionService (look at examples)
