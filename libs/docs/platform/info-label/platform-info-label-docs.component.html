Default Info Label Info Label With Text To apply semantic color to the Info Label use the [color] property. Available options are numbers from 1 to 10. Info Label With Text and Icon Set the [type] property to 'icon' for Info Label with text and icon. Info Label With a Number or an Icon Set the [type] property to 'numeric' for Info Label with a number. For Info Label with icon only set the property to 'icon'. Info Label With a Aria Label for accessibility. Set the [ariaLabel] and [ariaLabelledBy] property to 'user defined string', Info Label which allows user to add browser readablity to the respective component .
