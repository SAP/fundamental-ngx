Basic Product Switch inside Shellbar. Product switch with enabled Drag And Drop. The DnD functionality can be switched off by passing [dragAndDropEnabled]="false" to component. The event thrown on drop can be handled by binding to (productsChange). The element can stick to it's position, it can be achieved by passing stickToPosition: true value to the interface object. Example of usage is in Home product. Use (isOpenChange) to listen for open/close state changes. Large Product switch is displayed with a maximum of 4 columns on large desktop screens. Medium If there are 6 or fewer items to display, it is recommended to use the 3-column layout (medium). The component automatically switches to 3 columns when it detects fewer than 7 products. List Mode For narrow screens the Product Switch transforms to a List by itself. Add the [forceListMode]="true" to force the list layout, even on larger screens.
