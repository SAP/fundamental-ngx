<fd-tree> There are two different types of tree cells you can use, the 'simple' cells and the 'rich' cells. Simple cells can only display text. Rich cells can contain plain text or links. The tree component requires only two root arguments, a 'headers' array and a 'treeData' array. The children of the tree are generated recursively from data within the treeData array.

In a tree with simple cells, the treeData property is an array of objects that contain two properties, 'rowData' and optionally 'children'. Simple cell rowData is an array of strings, each string being a cell in the tree. The 'children' property is an array of objects that match this object's structure.
