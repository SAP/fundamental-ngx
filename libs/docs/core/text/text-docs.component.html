Basic Text Display text content with the text component. Line Clamping Limit the number of visible lines using the maxLines input. Expandable Text Add expand/collapse functionality with "more"/"less" links. Requires maxLines to be set. Use two-way binding with [(isCollapsed)] to control or track the state. Whitespace Preservation Preserve whitespace and line breaks using the whitespaces input. Hyphenation Control word breaking with the hyphenation input. Supports 'none', 'manual', and 'auto'. Learn more at MDN: CSS hyphens.
