SAP Icons SAP-icons is the default font of the component. No need to set the font input property.
It is recommended to add `aria` attributes to the icon component to add context for screenreader users. BusinessSuiteInAppSymbols Icons For BusinessSuiteInAppSymbols icon font you need to set the font input property to font="BusinessSuiteInAppSymbols" TNT Icons For SAP-icons-TNT icon font you need to set the font input property to font="SAP-icons-TNT"
