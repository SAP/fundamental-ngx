Semantic Levels Headings have 6 semantic levels (H1-H6). Use only one H1 per page and maintain ascending order without skipping levels (e.g., H1 → H2 → H3). The component automatically detects the semantic level from the element tag. Visual Styling Use [headerSize] to style headings differently from their semantic level, maintaining proper document structure while achieving design requirements. Text Truncation By default, title text truncates with ellipsis when it exceeds its container width. Text Wrapping Use [wrap]="true" to wrap overflowing text instead of truncating.
