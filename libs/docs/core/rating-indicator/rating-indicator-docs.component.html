Rating indicator examples The Rating Indicator component displays a star-based rating system. Users can view and interact with ratings by clicking on stars or using keyboard navigation. Use [value] to set the rating value, [indicatorCapacity] to set the maximum number of stars (default is 5), and [allowHalves] to enable half-star ratings. For two-way binding with forms, use [(ngModel)]. Rating indicator with displayMode Use [displayMode]="true" to show the rating in a non-interactive, read-only state. This is useful for displaying existing ratings without allowing user interaction. Custom icons Customize the rating indicator appearance by using [ratedIcon] and [unratedIcon] properties to specify custom icon classes. Rating indicator sizes Control the rating indicator size using the [size] property. Available sizes: 'xs' | 'sm' | 'md' | 'lg' | 'cozy' | 'compact' | 'condensed'. The default size is 'md'. Combine with [allowHalves]="true" to enable half-star increments for more precise ratings. Rating indicator with ratings data Display aggregate rating data using the [ratings] property with a key-value object where keys are rating values and values are vote counts. Use [displayAllRatings]="true" to show a popover with the breakdown of all ratings. Alternatively, use [ratingAverage] and [totalRatings] to display summary statistics. Playground Area
