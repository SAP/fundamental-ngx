Object Page
The object page floorplan is used to display and categorize all relevant information about an object. Categorized content can be accessed quickly using anchor or tab navigation.

Object page floorplan can be created by using the Dynamic Page component in conjunction with the Facets component in its header.

The header of the object page is collapsible, which helps the users focus on the actual page content but still ensures that important header information and actions are readily available.

The content area of the object page is scrollable and provides an information about the object. Components used there may vary depending on the object data.
