Button Types The button component follows fundamental-styles for button guidelines. The default button type is "Default button", no modifier is required. Use the [fdType]="" property to set the button type. Menu Button In order to use menu button, use [fdMenu]="true" Button Sizes Use [fdCompact] directive to explicitly set button size. Button with Icons Use [glyph]="'{{ '{' }}icon-name}'" to add an icon to the button. Button State 'disabled' and 'toggled' states can be achieved through native 'disabled' / 'aria-disabled' attributes and 'toggled' input property. Regarding the disabled buttons there are 2 flavours reflecting two opinions how to make a button disabled. Feel free to check Mozilla Doc for disabled attribiut and for aria-disabled. Toggled Button A toggle button switches between two actions. One of the actions is always active, one is inactive. Use the toggle button for secondary actions. Use toggled input property to set the action to active. The inputs for button will be translated into classes attributed to the fundamental-styles guidelines.
For example:
if positive is passed for fdType, the fd-button--positive class will be added to the element.
if compact is passed for size, the class fd-button--compact will be added to the element.
