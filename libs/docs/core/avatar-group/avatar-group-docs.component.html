Avatar Group Standard Usage

In this example, which is called individual type avatar group, which means that every item is displayed separately, every item can be focused separately and details template will be displayed for that item.

Group Type Standard Usage

The fd-avatar-group supports group type as well. In this case, all items are displayed as one item. This is useful when there is a need to display a group of avatars in a limited space. In this case, entire container can be focused and details template will be displayed for all items

Vertical Avatar Group Usage

The fd-avatar-group supports vertical orientation as well. In this case, all items are displayed in a vertical list and overflow is not supported.
