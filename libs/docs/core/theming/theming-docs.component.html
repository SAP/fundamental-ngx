Theme switcher Css files are switched via choosing option. Setting theme via url

Theming service allows to automatically set theme based on provided URL query parameter.

Parameter name can be configured through ThemingModule.

See example by adding ?customQueryParam=sap_fiori_3_dark to URL.

To make it work properly, you need to have imported RouterModule into the project.

 Using custom configuration

Developers can easily configure ThemingService and provide custom configuration object to manipulate the behaviour of ThemingService.

For more information please refer to ThemingConfig class available in the API section.

 Using custom themes Developers can easily define set of custom themes by providing custom configuration object for the ThemingModule.
