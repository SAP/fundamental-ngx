Also item list can be marked as selected, by passing [selected]="true". Unread List item

Byline list item can be marked as unread.

For this, pass [unread]="true" input property for the list item component.

Additionaly, list item can render unread indicator.

To display unread indicator, developers should pass [unreadIndicator]="true" input property to the list component.
