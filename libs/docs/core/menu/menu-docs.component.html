Simple menu To create simple menu use following directives:
  • fd-menu-item - list element
  • fd-menu-interactive - interactive element wrapping menu element content
  • fd-menu-title - menu element title
Attach created menu to any HTML element using fdMenuTrigger directive. Menu with icons Menu items can be extended with additional content using the following directives:
  • fd-menu-addon - adds icon to menu item. If you want to add the icon before the menu item title, remember to add the position="before"
  • fd-menu-shortcut - adds information about useful shortcuts
 Menu with separator To add a visual separation line between menu items, use the [hasSeparator] input on the list item component. Menu with Scrollbar Menu handles overflow using fd-scrollbar Menu with submenu Each Menu item can have its submenu. To create a submenu, do the following:
  • Create standard menu
  • Create submenus using fd-submenu component
  • Attach created submenus to fd-menu-item using [submenu] input
  • You can use fd-menu-addon with [submenuIndicator]="true" to add icon indicating menu items with submenus
Use (activePath) output to receive information about currently active items. Use (openOnHoverTime) to specify after how much hover time menu item should expand. Extended menu

Menu can have menu items as toggle button or as a segmented button. Menus also can include inputs as an interactive element.

To use toggle button functionality, use button[fd-menu-interactive][toggle] element. It implements control value accessor, so it can be used with both template-driven and reactive forms.

To use segmented button functionality, use li[fd-menu-item][fdMenuSegmentedButtonHeader] and li[fd-menu-item][fdMenuSegmentedButtonOption] elements.
li[fd-menu-item][fdMenuSegmentedButtonHeader] implements control value accessor, so it can be used with both template-driven and reactive forms. The possible values for it are passed via li[fd-menu-item][fdMenuSegmentedButtonOption][value] property and can be anything.
By default, the comparison is done using strict equality operator, but you can provide your own valueComparator to the header element. The type for it is (a, b) => boolean

To use input as an menu item, use input[fd-form-control][fd-menu-item-input] element inside the div[fd-menu-interactive] element.

Menu in mobile mode To use Menu in mobile mode:
  • Set [mobile] property to true
  • Set [mobileConfig] with MobileModeConfig configuration object to customize the look of the control
Menu Mobile Mode can be configured globally by providing MOBILE_MODE_CONFIG token. Check Global Configuration page for more information.
