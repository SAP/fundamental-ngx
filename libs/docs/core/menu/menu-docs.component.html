Simple menu To create simple menu use following directives:
  • fd-menu-item - list element
  • fd-menu-interactive - interactive element wrapping menu element content
  • fd-menu-title - menu element title
Attach created menu to any HTML element using fdMenuTrigger directive. Menu with icons Menu items can be extended with additional content using the following directives:
  • fd-menu-addon - adds icon to menu item. If you want to add the icon before the menu item title, remember to add the position="before"
  • fd-menu-shortcut - adds information about useful shortcuts
 Menu with separator and compact mode To add a visual separation line between menu items, use the fd-menu-separator component.
To display Menu in compact mode use fdCompact. Menu with Scrollbar Menu handles overflow using fd-scrollbar Menu with submenu Each Menu item can have its submenu. To create a submenu, do the following:
  • Create standard menu
  • Create submenus using fd-submenu component
  • Attach created submenus to fd-menu-item using [submenu] input
  • You can use fd-menu-addon with [submenuIndicator]="true" to add icon indicating menu items with submenus
Use (activePath) output to receive information about currently active items. Use (openOnHoverTime) to specify after how much hover time menu item should expand. Menu in mobile mode To use Menu in mobile mode:
  • Set [mobile] property to true
  • Set [mobileConfig] with MobileModeConfig configuration object to customize the look of the control
Menu Mobile Mode can be configured globally by providing MOBILE_MODE_CONFIG token. Check Global Configuration page for more information.
