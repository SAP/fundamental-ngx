Default Fixed Card Layout Auto arrangement of cards in fixed card layout. The layout is responsive and cards gets re-arranged when window resize happens or some cards gets hidden or shown. Disabling drag and drop Behavior of dragging cards can be disabled and enabled. Mobile display In mobile display as well layout is consistent and on changing orientation, cards will re-distribute and arrange itself. Custom card minimum width

By default, calculations for the amount of the available columns takes into consideration minimum card width, which is by default equals 320px (20rem).

This parameter can be changed by providing [cardMinimumWidth] input property. The value should be a number of pixels greater than 320.

Custom columns width

By default, columns are equal by their width. It can be changed by providing [columnsWidthConfig] input property. It works similar to the flex-grow in CSS. You should pass an object with the column indexes and their "flex-grow" factors.

Maximum number of columns

By default layout can have a maximum of 10 columns. This parameter can be changed by providing maxColumns input property.
