Generic Tag

The Generic Tag component displays complementary information related to the current page, such as key performance indicators (KPI) and situations. It combines a semantic decorator, KPI name, optional semantic message icon, and KPI value in a compact, accessible layout.

Component Structure

  • Semantic Decorator: Visual decoration designating the message status of the control
  • GenericTag Container: The layout container carrying all interaction states
  • KPI Name: Standard text displaying the key performance indicator name (keep it simple, use no more than 3 words)
  • Semantic Message Icon: Optional status icon (error, success, warning, information)
  • KPI Value: The numeric (key) attribute and its unit

Key Properties

  • type: Semantic status - 'error' | 'success' | 'warning' | 'information'. Omit for default style.
  • name: Required. The KPI name displayed as standard text.
  • value: Optional. The KPI value representing the numeric attribute and its unit.
  • ariaRoleDescription: Accessible role description (defaults to 'Generic Tag').
