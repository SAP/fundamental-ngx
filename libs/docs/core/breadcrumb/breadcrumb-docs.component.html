The breadcrumb is responsive. When the last breadcrumbs item's edge overflows the available area, the earliest breadcrumb will be added to an overflow menu at the start of the breadcrumb trail. By default, these items are displayed in the menu with the most recently added at the top. This can be changed by setting the [reverse] input to [true].
