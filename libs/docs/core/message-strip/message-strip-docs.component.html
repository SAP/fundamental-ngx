Message Strip Types A number of message strip types are available. They offer preset styling for different contextual situations. Can be one of 'warning', 'success', 'information', 'error' or normal (default). Message Strips Without Icons The default message strip has no icon. Information, Success, Warning and Error message strips by default are rendered with icons. To remove them set the [noIcon] property to true. Message Strips With Customizable Width The width of the message strip can be customized by the [width] and [minWidth] properties. Message Strip Alerts Message Strip Alerts are used to display a short, important message in a prominent location on the page. They are used to inform users of a situation that requires their attention or action. They are not part of the Fiori Design guidelines. {{ data.properties.message }}
