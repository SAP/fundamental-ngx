Basic Layout Panel Layout Layout Panel Edge Bleed Use [bleed]="true" on the panel body to have the contents of the panel body bleed to the outer edge. This is useful for components like tables and trees. Layout Panel Transparent For Layout Panel with transparent background set the [transparent]property of fd-layout-panel to true. Layout Panel Footer Variations By default, the content of the footer is rendered in the center. Use the position property of the fd-layout-panel-footer to place the content of the footer to the left (start) or to the right (end).
