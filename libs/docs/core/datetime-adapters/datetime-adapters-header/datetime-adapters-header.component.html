DateTime Adapters
Components such as Date Picker, Time Picker, and DateTime Picker rely on a DatetimeAdapter and DateTimeFormats to parse, format, and manipulate dates. Two adapters are available out of the box:

FdDate (built-in) Dayjs
Package @fundamental-ngx/core/datetime @fundamental-ngx/datetime-adapter
Custom parse formats Limited (native Date parsing) Full format-string control
Locale support Via browser Intl API dayjs locale plugins
UTC / strict mode No Yes
Best for Simple apps, zero-dep setups i18n-heavy or format-sensitive apps

Migrating from Moment? The @fundamental-ngx/moment-adapter package is deprecated because Moment.js is now considered legacy. See the Migration from Moment section below.
