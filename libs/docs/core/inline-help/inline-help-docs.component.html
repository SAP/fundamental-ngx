Inline Help Inline help component can be placed anywhere, by adding directive fd-inline-help with text to display.

Inline Help with Changed Trigger Event To change trigger event, same as in popover component, you need to pass events to [triggers] input.

Inline Help with Additional Class. Additional class can be added to the body of inline help, by passing additionalBodyClass input.

Inline Help with Template Inside. Instead of text, there is a way to place template inside inline help, by passing template to [fd-inline-help-template] input.
