Links

The Link component provides a semantic and accessible way to create hyperlinks with enhanced styling and functionality. It supports standard anchor tag features like href and integrates seamlessly with Angular Router.

Key Features

  • Visual Modifiers: Multiple style variations including emphasized, subtle, inverted, and undecorated modes
  • Icon Support: Automatically detects and positions icons projected as content, placing them as prefix or postfix elements
  • Accessibility: Built-in ARIA support with screen reader announcements for emphasized and subtle states
  • States: Handles disabled state with proper ARIA attributes and visual feedback
  • Touch Target: Optional enlarged touch target for improved mobile usability
  • Router Integration: Works with Angular RouterLink directive for SPA navigation
  • Flexible Content: Accepts any content including text, icons, or custom elements via content projection

Available Inputs

  • emphasized - Displays the link with emphasized styling and announces it to screen readers
  • subtle - Applies subtle styling with reduced visual prominence and screen reader announcement
  • inverted - Inverts the color scheme for use on dark backgrounds
  • disabled - Disables the link interaction and applies disabled styling with proper ARIA attributes
  • undecorated - Removes the default underline decoration
  • touchTarget - Enlarges the clickable area for better mobile accessibility
