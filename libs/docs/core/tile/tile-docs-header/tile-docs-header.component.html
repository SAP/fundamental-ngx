Tile
Tiles are container components used to display concise, relevant information in a structured and visually appealing way. They are commonly used in dashboards and home screens to represent applications, KPIs, or other content similar to the SAP Fiori launchpad.

Tiles can display various types of content:
  • Launch tiles for application shortcuts
  • KPI tiles showing key performance indicators with numeric content
  • Feed tiles for notification updates
  • Generic tiles with custom content including icons, titles, text, counters, and charts


Tile as Link or Interactive Element

Tiles support two usage patterns based on your needs:

  • Custom Element: Use <fd-tile> for interactive buttons or cards. Enable interactivity with [clickable]="true" and listen to (tileClick) events.
  • Native Link: Use <a fd-tile href="..."> to create tiles that act as native hyperlinks. This approach provides native browser behavior including right-click context menus, keyboard shortcuts, and better accessibility. All standard anchor attributes work: href, target, rel, hreflang, etc.


Accessibility

Tiles support comprehensive accessibility features. Use the following inputs to enhance accessibility:

  • id - Unique identifier for the tile
  • role - ARIA role (automatically set for native links)
  • ariaLabel - Descriptive label for screen readers
  • ariaLabelledby - Reference to element(s) that label the tile


Tile Building Blocks
  • fd-tile or [fd-tile]: The main tile container component.
    • size - Size modifier: 's' for small tiles on smaller screens (default: large)
    • double - When true, doubles the tile width to 2×1 dimensions
    • type - Tile type: 'kpi', 'launch', or 'feed'
    • action - When true, places tile in action/edit mode with overlay
    • clickable - Makes the tile focusable and clickable (for custom elements only)
  • fd-tile-header directive - Container for the tile header section
  • fd-tile-header-content directive - Wraps header content, can use [twoColumn]="true" for 2-column layout
  • fd-tile-title directive - Displays the main tile title
  • fd-tile-subtitle directive - Displays subtitle text
  • fd-tile-content directive - Main content area of the tile
  • fd-tile-content-text directive - Text content within the tile, can use [twoColumn]="true" for 2-column layout
  • fd-tile-footer directive - Container for tile footer
  • fd-tile-footer-text directive - Footer text content, can use [twoColumn]="true" for 2-column layout
  • fd-tile-logo directive - Tile logo image (32×32px recommended)
  • fd-tile-profile-img directive - Profile image within tile
  • fd-tile-background-img directive - Background image for tile
  • fd-tile-refresh directive - Refresh indicator with optional icon. Use glyph="icon-name" to customize the icon
  • fd-tile-section directive - Wraps a section of tile content
  • fd-tile-page-indicator directive - Page indicator for multi-page tiles
  • fd-tile-toggle directive - Toggle control within tile
  • fd-tile-action-container directive - Container for action mode controls
  • fd-tile-action-close directive - Close button for action mode (auto-adds icon)
  • fd-tile-action-indicator directive - Action indicator button (auto-adds icon)
  • fd-tile-title-container directive - Container for complex title layouts


Numeric Content (KPI Tiles)

KPI tiles use numeric content directives to display key performance indicators with values, scales, and trend indicators:

  • fd-numeric-content directive - Container for numeric content with scale and value. Supports state ('positive', 'negative', 'critical', 'informative') and size ('s', 'm', 'l') inputs
  • fd-numeric-content-kpi-container directive - Container for KPI value and unit
  • fd-numeric-content-kpi directive - The actual KPI numeric value
  • fd-numeric-content-scale-container directive - Container for scale elements
  • fd-numeric-content-scale directive - Scale value indicator with optional state ('positive', 'negative', 'critical', 'informative')
  • fd-numeric-content-scale-text directive - Text label for scale
  • fd-numeric-content-scale-arrow directive - Directional arrow indicator with optional state
  • fd-numeric-content-launch-icon-container directive - Container for launch icon
  • fd-numeric-content-launch-icon directive - Launch icon within numeric content
