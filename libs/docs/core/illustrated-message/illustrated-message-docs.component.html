Scene The configuration object passed to [svgConfig] should contain values for both 'scene' and 'dialog'. For screens smaller than 600px the 'dialog' svg is applied. Dialog Set the type property to dialog for Illustration Message used in a Dialog or a Message Box. Spot Set the type property to spot for Illustration Message used in medium-sized cards or UI elements smaller than full screen. Inline SVG

In some cases it's not applicable to use URLs for the SVG locations.

In this case developers can load SVG directly into their js code and pass it as a string to the component.
