Large The configuration object passed to [svgConfig] should define values for all sizes: 'large', 'medium', 'small' and 'xsmall'. The appropriate size is applied automatically based on the container's width. Medium Set the type property to medium for Illustration Message used in a Dialog or a Message Box. The configuration object passed to [svgConfig] should contain values for 'medium', 'small' and 'xsmall'. Small Set the type property to small for Illustration Message used in medium-sized cards or UI elements smaller than full screen. The configuration object passed to [svgConfig] should contain values for 'small' and 'xsmall'. Extra Small Set the type property to xsmall for Illustration Message used in small-sized cards or UI elements smaller than full screen. The configuration object passed to [svgConfig] should contain values for 'xsmall'. Inline SVG

In some cases it's not applicable to use URLs for the SVG locations.

In this case developers can load SVG directly into their js code and pass it as a string to the component. The configuration object passed to [svgConfig] should contain values for 'large', 'medium', 'small' and 'xsmall'.
