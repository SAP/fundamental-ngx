Illustrated Message

Illustrated messages for SAP Fiori are a simple yet powerful way to engage users and add personality to the experience.


Structure

  • Illustration - it helps clarify the situation and must always be paired with a message.
  • Message - fd-illustrated-message-figcaption
    Consists of:
    • Headline - fd-illustrated-message-title - a single line and explains the state.
    • Description - fd-illustrated-message-text - adds details and, where appropriate, tells the user what to do next. It's recommended to fit the text in three lines or less.


  • Button (optional but recommended) - fd-illustrated-message-actions
    Include a call to action (CTA) button if there is a clear next step.

Responsiveness and Adaptiveness

Illustrated Message Update: The illustration types have been renamed from scene, dialog, spot, and dot to large, medium, small, and extra small for clarity and consistency. Legacy type values and modifier classes are still supported for backward compatibility.

Resizing is not required or recommended for Medium and Small illustrations. Large illustrations automatically scale based on the width of their container. The container uses breakpoints to adjust the size responsively, as shown below:
