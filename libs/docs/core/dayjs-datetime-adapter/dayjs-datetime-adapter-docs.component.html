Date Picker example with DayjsDatetimeAdapter DateAdapter is a generic class, components such as Datepicker also need to be made generic. When working with these classes (for example as a ViewChild) you should include the appropriate generic type that corresponds to the DateAdapter implementation you are using. For example:

@ViewChild(DatePickerComponent) datePicker: DatePickerComponent<Dayjs>; Locale time zone and strict mode By default the DayjsDatetimeAdapter creates dates in your time zone specific locale and in without strict parsing. You can change the default behaviour by setting useUtc: true and strict: true. DayjsDatetimeAdapter formats The DATE_TIME_FORMATS object is a collection of formats that components such as datepicker, timepicker, etc. uses when parsing and displaying dates. These formats are passed through to the DatetimeAdapter. The DayjsDatetimeAdapter has DAYJS_DATETIME_FORMATS, provided in the code of the example.

These formats could be customized, but be sure they are compatible with the DatetimeAdapter.
