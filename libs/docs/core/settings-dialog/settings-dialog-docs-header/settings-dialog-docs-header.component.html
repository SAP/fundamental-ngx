Settings Dialog

The Settings Dialog allows users to personalize their Fiori experience by adjusting various settings. It follows a standard dialog layout, with a split content area. The left section displays a list of option groups, and selecting an option shows the corresponding settings in the detail area.

The Settings Dialog is a composite component that includes the following elements with additional directives and properties:

  • Dialog: In the dialog configuration object, add settings: true. This applies the necessary classes to the dialog and its components, giving it the appearance of a settings dialog.
  • Bar
  • Title
  • Input Group
  • List: includes boolean propertied settingsList and settingsListFooter.
  • List Item: includes an input property settingsListTpl, which links a list item to a template rendered in the Settings Detail area.
  • Button: includes the additional fd-settings-header-button directive. This button is rendered in the Detail area and is visible only in mobile modes (sm and md). It serves as a back navigation to the List area and is a required element.
  • Icon Tab Bar: includes a boolean property settings.
