Tool Layout
The Tool Layout is an essential part of most BTP tools. The layout combines the Tool Header and the Navigation components in one consistent structure.

Anatomy

  • Tool Header - always present and is a top-aligned UI element that contains global actions and controls navigation
  • Navigation - side navigation in all of its variants (expanded, snapped)
  • Content - an empty container area that can hold any content.

Structure

  • fdbToolLayout - For tablet mode set the input property mode to 'tablet'. For phone/mobile mode set the input property mode to 'phone'. The default mode is desktop and doesn't have to be set.
  • fdbToolLayoutContainer
  • fdbToolLayoutHeaderContainer
  • fdbToolLayoutNavigationContainer - For a navigation in popup set the input property popup to true. If you need horizontal navigation set horizontal input property to true.
  • fdbToolLayoutContentContainer - For transparent background set the input property background to 'transparent'. For list background set the input property background to 'list'.
