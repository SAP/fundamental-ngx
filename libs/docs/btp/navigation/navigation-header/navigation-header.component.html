Navigation

The Side Navigation is the only option to add a navigation to an application. It offers the user easy access to multiple levels.

States

Expanded

  • The first levels are a collapsible group (navigation group) â€“ no icon
  • If second-level items have children, they show the expand/collapse arrows
  • Second-level items display an icon and are not indented
  • If second-level items have children, they show the expand/collapse arrows
  • Third-level items do not display an icon and are indented by 2 rem (32 px)

Snapped (collapsed)

  • The first levels always show an icon
  • If first-level items have children, they show the expand/collapse arrows
  • Second-level items do not display an icon and are not indented
  • If second-level items have children, they show the expand/collapse arrows

The Side Navigation Button placed in Shell Bar allows switching between expanded and snapped(collapsed) modes.

Popup (overlay)

In this mode the Side Navigation is placed inside a Popover component. The trigger is the side navigation button placed in Shell Bar. On small screen sizes (600px) it should follow the behaviour of the Responsive Popover.

Modes

  • Desktop (compact)
  • Tablet (cozy)
  • Phone (cozy)
