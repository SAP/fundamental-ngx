Available languages:

Import any of these: FD_LANGUAGE_ENGLISH, FD_LANGUAGE_SPANISH, FD_LANGUAGE_FRENCH, FD_LANGUAGE_GERMAN, FD_LANGUAGE_CHINESE, FD_LANGUAGE_JAPANESE, FD_LANGUAGE_POLISH, FD_LANGUAGE_UKRAINIAN, and 20+ more!

Where to find translation keys:

fundamental-ngx provides pre-translated strings for all its components. You can discover available translation keys in several ways:

  • Component documentation: Each component has an i18n section listing all its translation keys. For example, see Date Picker translation keys.
  • TypeScript autocomplete: Type 'coreDatePicker.' in your IDE and autocomplete will show all available date picker translations like dateInputLabel, dateRangeInputLabel, etc.
  • FdLanguage interface: Explore the FdLanguage type definition to see all available keys across all components.
Common translation keys:
  • coreDatePicker.dateInputLabel - "Enter date"
  • coreFileUploader.browse - "Browse"
  • platformTable.emptyTableMessage - "No data"

  3. Use () to invoke the signal and get the actual string value
  4. The translation automatically updates if the language changes!
